Q-109.660
Konfiguration zur Nutzung von Message Tracking


Frage
Problem Beim internen sowie externen Versand von eMails soll überprüft werden, was mit versendeten eMails geschehen ist, z.B. ob der Empfänger die Nachricht gelesen oder gelöscht hat. Diese Funktion übernimmt in David das »Message Tracking«. Welche Einstellungen sind zur Nutzung des Message Trackings erforderlich?
Produkt David
Priorität normal
Effekt normal
Datum 26.02.2010

Antwort
Das Message Tracking dient der Nachverfolgung versendeter eMails, um zu erfahren, was mit der Nachricht beim Empfänger geschehen ist (z.B. empfangen, gesehen, verschoben, gelöscht). Zur Nutzung des Message Tracking müssen folgende Voraussetzungen erfüllt sein:
  • Der Absender der Nachricht muss Message Tracking nutzen dürfen und diese Funktion entweder generell oder separat für die betreffende Nachricht aktiviert haben.
  • Der Empfänger der Nachricht muss das Zurücksenden von Tracking-Informationen erlauben (z.B. Einstellung Tracking-Informationen übermitteln im David Client).
Konfiguration und Nutzung von Message Tracking

Die folgenden Punkte erläutern die erforderliche Konfiguration sowie Nutzung von Message Tracking. Beachten Sie außerdem den KnowledgeBase Artikel Q-100.537 sowie die Hinweise zur Konfiguration und Nutzung von Message Tracking:

  • Konfiguration im David.Administrator
  • Versand von »getrackten« Nachrichten
  • Abrufen von Tracking Informationen

Konfiguration im David.Administrator

Um das Message Tracking generell nutzen zu können, muss diese Funktion in der Systemkonfiguration des David.Administrator aktiviert werden. Klicken Sie dazu im Konfigurationsbaum mit der rechten Maustaste auf den Punkt »System«, wählen Sie im Kontextmenü den Eintrag »Konfigurieren« und öffnen Sie im zugehörigen Konfigurationsfenster die Registerkarte »Optionen«. Stellen Sie sicher, dass hier die Option »Message Tracking erlauben« aktiviert ist. Speichern Sie Ihre Änderungen mit einem Klick auf »OK« ab.

Versand von »getrackten« Nachrichten

Öffnen Sie den David Client, um festzulegen, ob Sie das Message Tracking beim Versand von eMails generell oder selektiv aktivieren möchten. Wechseln Sie dazu über die Menüleiste im David Client in den Bereich »Optionen > Einstellungen > Global«. Aktivieren Sie hier die Option »Nachrichten grundsätzlich verfolgen«, um das Message Tracking generell für den Versand aller eMails zu aktivieren. Die Option »Tracking-Informationen übermitteln« können Sie optional aktivieren, wenn Sie die Übertragung von Tracking-Informationen an die Absender »getrackter« empfangener Nachrichten erlauben wollen. Speichern Sie anschließend Ihre Änderungen mit einem Klick auf »OK« ab.

Wollen Sie das Message Tracking lediglich selektiv für bestimmte eMail-Sendeaufträge nutzen, so achten Sie darauf, dass die Option »Nachrichten grundsätzlich verfolgen« deaktiviert ist. Im Nachrichteneditor können Sie das Message Tracking dann manuell aktivieren. Gehen Sie dazu folgendermaßen vor:

  • Öffnen Sie den Nachrichteneditor und klicken Sie in der Symbolleiste »Optionen« auf die Message Tracking Schaltfläche (Lupen-Symbol mit grünem Häkchen). Dadurch wird diese Funktion aktiviert.
    Falls diese Schaltfläche nicht sichtbar ist, passen Sie die Symbolleiste an, indem Sie mit der rechten Maustaste darauf klicken und aus dem Kontextmenü den Eintrag »Anpassen« auswählen. Im darauf erscheinenden Fenster können Sie unterhalb von »Nachrichten-Editor > Optionen« die Anzeige der »Message Tracking« Schaltfläche aktivieren. Bestätigen Sie Ihre Änderungen mit einem Klick auf »OK«.

  • Alternativ dazu können Sie das Message Tracking in den Nachrichtenoptionen einer eMail aktivieren. Klicken Sie dazu in der Menüleiste des Nachrichteneditors auf »Datei > Nachrichtenoptionen« und wählen Sie auf der Registerkarte »Erweitert« die Option »Message Tracking«. Bestätigen Sie diese Einstellung mit einem Klick auf »OK«.

Abrufen von Tracking Informationen

Um Tracking Informationen im David Client aufzurufen, stehen Ihnen zwei Möglichkeiten zur Verfügung:

  1. Wechseln Sie in Ihren Ausgangsordner und markieren Sie eine versendete Nachricht mit der linken Maustaste. Im Vorschaufenster der Nachricht steht Ihnen eine zusätzliche Registerkarte mit Tracking Informationen in Kurzform zur Verfügung. Für nähere Informationen klicken Sie auf diese Registerkarte. (Diese Funktion steht ab David.fx zur Verfügung.)

  2. Wechseln Sie in Ihren Ausgangsordner, markieren Sie in der Eintragsliste eine versendete eMail mit der rechten Maustaste und rufen Sie im Kontextmenü den Eintrag »Tracking Informationen« auf. Im darauf erscheinenden Fenster werden Ihnen die Informationen angezeigt.
