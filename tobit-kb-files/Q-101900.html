Q-101.900
Installation und Einrichtung von Tobit Mobile DvISE


Frage
Problem Was ist bei der Installation und Einrichtung von Tobit Mobile DvISE zu beachten?
Produkt David.InfoCenter Mobile
Priorität normal
Effekt normal
Datum 17.03.2005

Antwort
Installation
Die Installation von Tobit Mobile DvISE gliedert sich grundsätzlich in zwei Abschnitte. Der erste Abschnitt beinhaltet die Installation der Server-Komponenten auf dem DvISE Server, im zweiten Abschnitt folgt die Installation und Einrichtung der Tobit Mobile DvISE Clients.

Um Tobit Mobile DvISE nutzen zu können, müssen Sie zunächst die notwendigen Server-Komponenten auf dem DvISE Server installieren. Diese Installation ist einmalig vor dem ersten Einsatz von Tobit Mobile DvISE erforderlich. Laden Sie zunächst den Programmcode aus dem Internet herunter. Sie finden das Download im Club Tobit unter club.tobit.com.

Hinweis:
Die Nutzung von Tobit Mobile DvISE ist bzw. war unter David 6.6 nur bei Einsatz der optional erhältlichen Lizenz (kostenpflichtige Erweiterungslizenz) möglich. Der Vertrieb der Lizenz wurde allerdings eingestellt und das Download steht somit nicht mehr zur Verfügung. Ab David XL ist das Tobit InfoCenter Mobile aber schon in der Startlizenz enthalten. Ein Erweiterungslizenz nicht dann hierfür nicht mehr notwendig.

Installation auf dem DvISE Server
Stellen Sie sicher, dass Sie als Benutzer mit Administratorrechten angemeldet sind, und führen Sie die Download-Datei unter Windows NT/2000/XP direkt auf dem DvISE Server aus bzw. bei einer Netware-Installation auf einer Arbeitstation mit Zugriff auf den DvISE Server. Daraufhin startet das Setup-Programm, das Sie automatisch durch den Installationsprozess führt.
Wählen Sie als Zielpfad für die Installation das Verzeichnis DAVID auf Ihrem DvISE Server aus. Falls bisher noch kein System-Passwort für die DvISE Replication Services vergeben wurde, werden Sie nun zur Eingabe eines Passworts aufgefordert. Das nachfolgende Fenster bietet die Möglichkeit, dieses Passwort für die anschließende Installation von Tobit Mobile DvISE auf den Arbeitsstationen zu speichern. In diesem Fall wird das Passwort dann bei der Einrichtung der Clients automatisch in die Konfiguration des Tobit InfoCenter XP (»Optionen -> Einstellungen -> Mobile«) übernommen und muss später nicht manuell eingetragen werden.
Beachten Sie jedoch, dass bei einer Speicherung des Passworts jeder Benutzer, der auf seiner Arbeitsstation über Administratorrechte verfügt, die Client-Komponenten von Tobit Mobile DvISE auf seiner Arbeitsstation installieren und nutzen kann, da ja ein Passwort bei der Installation nicht mehr abgefragt wird. Legen Sie anschließend fest, welche globalen Archives in die Replizierung einbezogen werden sollen und geben Sie schließlich den Domain-Namen bzw. die IP-Adresse des DvISE Servers an.

Einrichten der Arbeitsstationen
Nachdem die Server-Komponenten von Tobit Mobile DvISE auf dem DvISE Server eingerichtetsind, kann die Installation des Tobit InfoCenter XP auf den mobilen Arbeitsstationen vorgenommen werden. Diese müssen korrekt im Netzwerk angemeldet sein und Zugriff auf den DvISE Server haben. Für die Installation müssen Sie über Administratorrechte auf der jeweiligen Arbeitsstation verfügen.

Standardmäßig können bis zu zehn mobile Arbeitsstationen gleichzeitig ihre Daten mit demselben Host Server replizieren. Diese Maximalwert kann bei Bedarf in der Systemkonfiguration der DvISE Replication Services am DvISE Server verändert werden (Menüpunkt »Max. Verbindungen«).

Wechseln Sie von der Arbeitsstation aus über das Netzwerk in das \DAVID\CLIENTS Verzeichnis auf dem DvISE Server und führen Sie dort die Datei MOBILE.EXE aus, worauf das Installationsprogramm gestartet wird. Wählen Sie die gewünschte Dialogsprache und tragen Sie bei Aufforderung die beiliegende Lizenznummer ein. Diese finden Sie auf dem »License Certificate«, das sich im Lieferumfang von Tobit Mobile DvISE befindet. Beachten Sie, dass für jede der einzurichtenden mobilen Arbeitsstationen eine eigene Lizenznummer erforderlich ist.

Auf den Arbeitsstationen, auf denen Tobit Mobile DvISE installiert werden soll, darf weder das DvISE Backend noch eine Version des Tobit InfoCenter (XP) bereits installiert sein.

Wenn während der Server-Installation das System-Passwort für die Replizierung nicht gespeichert wurde, werden Sie nun zur Eingabe dieses Passwortes aufgefordert. Ohne die Eingabe des korrekten System-Passworts wird die Installation zwar zu ende geführt, eine Replizierung mit dem Host Server führt jedoch zu einer Fehlermeldung (Replikationsmonitor). Erst wenn im Tobit InfoCenter XP unter »Optionen -> Einstellungen« auf der Registerkarte »Mobile« das korrekte System-Passwort eingetragen wurde, erfolgt eine fehlerfreie Replizierung.
Zum Abschluss der Client-Installation schließlich erfolgt eine Abfrage, ob beim Hoch- bzw. Herunterfahren des Betriebssystems an eine Replikation erinnert werden soll. Zu diesem Zweck wird auf der mobilen Arbeitsstation ein Dienst installiert und gestartet. Setzen Sie den Schalter »Erinnerung einschalten«, wenn Sie an die Replizierung erinnert werden möchten.
Sofern für den Remote Access auf die Archives eines Benutzers noch keine Zugriffsparameter vergeben wurden, erfolgt beim ersten Start des Tobit InfoCenter XP die Abfrage dieser Parameter, die für die Durchführung der Replizierung benötigt werden.

Betrieb
Nach erfolgreicher Installation von Tobit Mobile DvISE sowohl auf dem DvISE Server als auch auf den mobilen Arbeitsstationen ist das System einsatzbereit. Auf den Arbeitsstationen ist nun eine spezielle Version des Tobit InfoCenter XP installiert, die sich jedoch in der Bedienung von der bisherigen Version nicht unterscheidet. Hinzugekommen sind lediglich unter dem Hauptmenüpunkt »Optionen - > Einstellungen« die Registerkarte »Mobile« sowie ein neues Icon in derSymbolleiste, mit der eine Replizierung mit dem DvISE Server manuell gestartet werden kann.
Im Menü »Netzwerk« kann über die Option »Offline« der Offline-Modus aktiviert bzw. deaktiviert werden. Des Weiteren finden Sie dort als neue Server-Einträge den Namen des Host Servers und den der Arbeitsstation. Über diese beiden Einträge kann ohne Replizierung zwischen Host Server und Arbeitsstation umgeschaltet werden. Der Wechsel zur Arbeitsstation entspricht dann dem Wechsel in den Offline-Modus.

Einstellungen im Tobit InfoCenter
Unter dem Hauptmenüpunkt »Optionen - > Einstellungen -> Mobile« können Sie Einstellungen für den Abgleich mit dem DvISE Server konfigurieren. Diese Registerkarte ist nur verfügbar wenn »Tobit Mobile DvISE« korrekt auf der Arbeitsstation installiert ist. Hier stehen die folgenden Konfigurationspunkte zur Verfügung:

  • Host IP/Domäne
    Tragen Sie hier die IP-Nummer des Host Servers oder den Domänen-Namen ein.
  • System Passwort
    Das hier eingetragene System Passwort gilt für die Replizierung Ihrer Daten mit dem Host-Rechner. Das Passwort darf nur verändert werden, wenn auch der gleichnamige Eintrag auf dem Host Server (Menüpunkt »DvISE Replica«) angepasst wird. Im Zweifelsfall belassen Sie den Eintrag unverändert, um Fehler bei der Replizierung zu vermeiden.
  • Beim Starten/Beenden des PCs replizieren
    Aktivieren Sie den Schalter, wenn beim Herauf- bzw. Herunterfahren des Betriebssystems eine Abfrage erfolgen soll, die Arbeitsstation mit dem Host Server zu replizieren.
  • Remote Access Parameter verwenden
    Setzen Sie diesen Schalter, wenn Sie wünschen, dass für die Archive-Replizierung dieselben Authentifizierungsparameter verwendet werden, wie für den Remote Access. Wenn Sie die Remote-Access-Parameter nicht verwenden möchten, tragen Sie Benutzername und Passwort in die zugehörigen Felder ein.
  • Intervall
    Wählen Sie aus der Liste ein Intervall, wonach automatisch eine Replizierung mit dem DvISE Server vorgenommen werden soll. Nach dem hier konfigurierten Zeitabstand wird generell versucht, einen Abgleich durchzuführen. Wählen Sie z.B. ein Intervall von 30 Minuten um zu erreichen, dass spätestens eine halbe Stunde, nachdem der Rechner wieder mit dem Netzwerk verbunden wurde, automatisch ein Abgleich stattfindet. Ist nach Ablauf des Intervalls keine Verbindung zum DvISE Server möglich, wird eine entsprechende Meldung imArchive »System\DvISE\Fehlermeldungen« erzeugt.

Offline-Modus aktivieren bzw. deaktivieren
Solange sich die mobile Arbeitsstation im lokalen Netzwerk befindet, also direkte Verbindung zum DvISE Server besitzt, greift das Tobit InfoCenter XP direkt auf diesen Server zu. Das bedeutet, dass sämtliche Aktionen im Tobit InfoCenter XP sofort über den DvISE Server abgewickelt werden. Änderungen an Inhalten im Tobit Archive System werden ebenfalls sofort auf dem Server vorgenommen.

Wenn der mobile Computer vom Netzwerk getrennt wird, ist der direkte Zugriff auf den Server natürlich nicht mehr möglich. Daher muss vor dem Trennen der Netzwerkverbindung der Offline-Modus aktiviert werden.

Klicken Sie dazu auf »Netzwerk -> Offline« in der Menüleiste des Tobit InfoCenter XP. Dieses wird nun intern auf die lokale Kopie des Tobit Archive Systems umgeschaltet. Im Normalfall sollte anschließend eine Archive-Replizierung durchgeführt werden, um Ihre Arbeitsstation und den DvISE Server auf den gleichen Stand zu bringen. Es erscheint ein Dialogfenster, das zur Durchführung der Replizierung auffordert.

Im Offline-Modus wird ausschließlich mit der lokalen Kopie des Tobit Archive Systems gearbeitet. Die während der Offline-Zeit durchgeführten Änderungen werden auf den DvISE Server übertragen, sobald die Arbeitsstation wieder mit dem Netzwerk verbunden und das Tobit InfoCenter XP wieder in den Online-Modus geschaltet wird. Das hat zur Folge, dass z.B. ein Versandauftrag, der im Offline-Modus erstellt wurde, erst bei der nächsten Replizierung an den Host Server übertragen und von diesem anschießend versendet wird. Arbeitet man mitder Arbeitsstation zwischenzeitlich wieder im Offline-Modus, so wird dieser Versandauftrag mit dem entsprechenden Status-Eintrag wiederum erst mit der nächsten Replizierung im Ausgangsbuch der Arbeitsstation sichtbar.

Klicken Sie dazu wiederum den Menüpunkt »Netzwerk -> Offline« an. Das Häkchen vor »Offline« verschwindet und das Tobit InfoCenter XP wird nun wieder für den direkten Zugriff auf den DvISE Server umgeschaltet. Auch hier erscheint das Dialogfenster, das zur Durchführung der Archive-Replizierung auffordert, um wiederum die Datenbestände beider Computer miteinander abzugleichen.

Weiterführende Informationen
Ausführliche Hilfetexte finden Sie zum einen im DvISE-Systemhandbuch »David Information Server Engine Systemhandbuch und technische Referenz« in Kapitel 13 (Replication Dienste), zum anderen im »Lehr- und Arbeitsbuch für das Tobit InfoCenter XP« in Kapitel 2.2.1 (Registerkarte „Mobile”) und 4.1.3 (Registerkarte „Dienste”). Außerdem steht für jedes Dialogfenster per [F1]-Taste eine Online-Hilfe zur Verfügung.

Hinweis:Bei der Nutzung von Tobit Mobile DvISE unter Windows XP ist es u.U. erforderlich, dass Sie die »Einfache Dateifreigabe« deaktivieren. Weitere Informationen entnehmen Sie dem KnowledgeBase Artikel Q-100.903.
