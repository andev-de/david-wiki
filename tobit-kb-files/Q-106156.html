Q-106.156
Digital Sender 9100C mit DvISE nutzen


Frage
Problem Wie kann der Digital Sender 9100c unter David bei einer Installation unter Novell NetWare oder Windows eingerichtet werden?
Produkt David
Priorität normal
Effekt harmlos
Datum 10.07.2006

Antwort

Hinweis zur Installation:
Die Schnittstelle zwischen David oder FaxWare und dem Digital Sender ist der Digital Sender Service, welcher auf einer Windows NT 4.0/2000/XP Workstation oder einem Windows NT/2000 Server installiert werden muss. Auf diesem Rechner müssen Sie zusätzlich das FaxGate von der aktuellen CD installieren. Der David oder FaxWare Service Layer kann auf einem Windows NT Rechner, oder auf einem Novell NetWare Server installiert werden.

  1. Digital Sender Hardware Installation

    Zu Beginn müssen Sie einige Grundeinstellungen wie IP Adresse, IP Gateway, etc. direkt am Digital Sender eingeben. Dafür drücken Sie an der Digital Sender Tastatur die folgende Tastenkombination: »SHIFT+ALT+STARTTASTE« (Grüne Taste). Nehmen Sie nun die entsprechenden Einstellungen vor. Diese Grundeinstellungen sind für die Einrichtung des Digital Senders als E-Mail Client wichtig.

  2. Digital Sender Software Installation

    Die zu installierende Software gehört zum Lieferumfang des Digital Senders und beinhaltet folgende Software-Komponente:

    - HP Enterprise Center für HP Digital Sender 9100C
    - HP Digital Sender Service
    - HP Address Book Manager
    - HP Digital Sender Link
    - HP Handbücher

  3. Nach dem Start wählen Sie den Pfad aus, in den die Software des Digital Sender installiert werden soll.

  4. Wählen Sie im Dialogfenster "Setuptyp" den Menüpunkt "Anpassen".

  5. Wählen Sie nun im Dialogfenster "Komponenten wählen" zumindest den Digital Sender Service aus. Alle weiteren Komponenten werden für den Betrieb des HP 9100 im Zusammenhang mit David nicht unbedingt benötigt.

  6. Wichtig ist, das die folgenden Voraussetzungen unbedingt eingehalten werden.
    • Die Installation wird nur für die Betriebssysteme Microsoft® Windows® NT 4.0 und Windows 2000 unterstützt.
    • Die Installation kann nur vom Systemadministrator oder einem Benutzer ausgeführt werden, der Mitglied der Administratorgruppe ist.
    • Für den Dienst sind ein Benutzername und ein Kennwort eines Windows NT-Kontos erforderlich, das über folgende Eigenschaften verfügt:
      • Das Kennwort des Benutzerkontos läuft nie ab.
      • Das Benutzerkonto gehört zur Administratorgruppe.
      • Die Berechtigungen des Benutzerkontos sind auf "Anmeldung als Dienst" eingestellt.
    • Hewlett-Packard empfiehlt, dass der Digital Sender-Dienst auf dem LAN-Fax-Server installiert wird, oder auf einem Computer, auf dessen Arbeitsbereichverzeichnis der LAN-Fax-Server zugreifen kann. Das LAN-Fax-Arbeitsbereichverzeichnis muss den Installationsanforderungen des gewählten LAN-Fax-Herstellers genügen.
    • Computer, die den HP Digital Sender-Dienst für LAN-Fax verwenden, können nicht gleichzeitig mit HP Network ScanJet 5 oder mit HP Digital Sender Link arbeiten.

  7. Geben Sie nun im Dialogfeld "Digital Sender 9100C-Dienst für LAN-Fax" den Benutzer an, der das Recht hat einen Dienst zu starten und über die nötigen Rechte auf dem Rechner verfügt. Dies ist meistens der Administrator.

  8. Bestätigen Sie die Frage ob die Faxdienstfunktion installiert werden soll mit "JA".

  9. Wählen Sie im folgenden Dialogfeld "David by Tobit Software" aus.

  10. Geben Sie nun im folgenden Dialogfeld das Verzeichnis an, in dem die Faxe abgelegt werden sollen.Dieses Verzeichnis müssen Sie später auch in der FaxGate Konfiguration angeben.

  11. Einrichtung des Digital Sender für E-Mail und Faxdienst mit dem WebAdmin

    Den WebAdmin erreichen Sie über die im Digital Sender eingestellte IP Adresse z.B. http://192.168.2.10:4244.

    Hier besteht die Möglichkeit alle Konfigurationen des Digital Senders vorzunehmen. Zunächst ist es wichtig, das die TCP/IP und DNS Konfigurationen unter dem Menüpunkt "Verbindungen" korrekt eingestellt werden.

  12. Einrichtung des Digital Sender für E-Mail mit dem WebAdmin

    Unter dem Menüpunkt "Senden an - E-Mail" können Sie die nötigen Konfigurationen für den SMTP Server vornehmen.Setzen Sie hierzu die IP Adresse des David Servers und die weiteren erforderlichen Daten ein.

  13. Einrichtung des Digital Sender für Fax mit dem WebAdmin

    Unter dem Menüpunkt "Senden an - Fax" können Sie die nötigen Konfigurationen für den Fax Server vornehmen.Setzen Sie hierzu die IP Adresse des Servers, auf dem der Digital Sender Service läuft ein.

  14. Fax Gateway für HP Digital Sender

    Seit dem David 5.2 wird eine neue FaxGate Version mitgeliefert. Sie können diese Version auf dem David CD im Verzeichnis »NT\DAVID\UTIL\WINDOWS\FAXGATE« finden. Diese Software muss auf einer Windows NT Rechner installiert werden, welcher Zugriff auf das HP Digital Sender Verzeichnis (\HPFSCAN) hat. Weiterhin muss der Client Schreibrechte im API-Verzeichnis des David bzw. FaxWare Servers haben.

  15. Installation des FaxGate

    Starten Sie das HP FaxGate durch Aufruf von FAXGATE. Auf Ihrem Client erscheint die folgende Meldung:

      DAVID SERVER NOT GIVEN
    Bestätigen Sie diese Meldung mit OK.
  16. Konfiguration von FaxGate

    In diesem Menü nehmen Sie nun die folgenden Einstellungen vor:

      David Server
      Tragen Sie hier den Servernamen ein, auf dem David oder FaxWare installiert ist. Wenn sich die Installation auf einem NetWare Server und hier noch auf einem anderen Volume befindet, so muss der Name des Volumes mit angegeben werden. Scan Interval
      Tragen Sie hier die Zeit in Sekunden ein, in denen die Verzeichnisse nach Aufträgen durchsucht werden sollen. HPFSCAN
      Tragen Sie hier den Pfad für das HPFSCAN Verzeichnis incl. Servernamen ein. Hier werden später alle vom Scanner verarbeiteten Fax-Jobs abgelegt. Die Angabe muss in der UNC Schreibweise erfolgen. Mit dem Kontrollkästchen SCAN können Sie das Scannen des Verzeichnisses aktivieren bzw. deaktivieren. Beispiel: \\DAVID\SYS\HPFSCAN oder C:\HPFSCAN Pre Include
      Include-Datei. Diese Datei kann beliebige FaxWare Befehle (Klammeraffen-Befehle) enthalten. Diese Datei wird am Anfang des JobFiles für David oder FaxWare von FaxGate automatisch eingebunden. Post Include
      Include-Datei. Diese Datei kann beliebige FaxWare Befehle (Klammeraffen-Befehle) enthalten. Diese Datei wird am Ende des JobFiles für David oder FaxWare von FaxGate automatisch eingebunden. E-Mail Attachments
      Geben Sie an, ob E-Mail Attachments im Format BMP oder TIFF/G4 versendet werden sollen. CPU Utilization
      Stellen Sie die CPU Auslastung des PCs durch Tobit FaxGate ein. Sie sollten die Auslastung über einen längeren Zeitpunkt beobachten und dann entsprechend Ihren Vorstellungen einstellen.

      Das FaxGate bindet sich jetzt als so genannte "SYSTEM-TRAY-APPLIKATION" in Ihre Windows-Taskleiste ein und wird zuerst inaktiv. Klicken Sie auf dem FaxGate Icon mit der rechten Maustaste und auf "ACTIVATE FAXGATE". Das FaxGate Icon ist jetzt aktiv. ACHTUNG: Diese Bindung dauert einige Sekunden. In dieser Zeit bleibt das Fenster FaxGate Konfiguration öffnen. Warten Sie bis dieses Fenster sich schließt.

  17. Faxe versenden
    Wählen Sie am Digital Sender den Punkt Fax aus.

    Anschließend geben Sie die Faxnummer ein und bestätigen Sie diese mit der Enter Taste.

    Um das Fax nun zu versenden drücken Sie die grüne Kopieren Taste.

    Zum senden drücken Sie abschließend den senden Button.
    Hinweis: Das FAXGATE Icon wird bei der Erfassung der Dateien aus dem Faxverzeichnis grün blinken.

  18. E-Mails versenden

    Wählen Sie am Digital Sender den Punkt eMail aus.

    Geben Sie im folgenden Dialog die Daten wie Absender und eMail Adresse des Empfänger ein.

    Um die eMail zu versenden drücken Sie die grüne Kopieren Taste.

    Zum senden drücken Sie abschließend den senden Button.

  19. Dokument in David Archivieren
    Sie müssen das gescanntes Dokumentes als Fax versenden. Statt eine Faxnummer einzugeben, tragen Sie Ihren David Benutzernamen in geschweiften Klammer ein {TTOBIT}. Dieses Dokument wird anschließend in Ihrem Eingangsbuch erscheinen.

    Weitere Informationen zu HP Produkten und die aktuelle Digital Sender Software finden Sie auf der Webseite von Hewlett-Packard.
