Q-106.693
Aufrufparameter für das InfoCenter (DVWIN32.EXE)


Frage
Problem In einer Branchensoftware sollen Nachrichten, die sich im Tobit Archive System befinden, mit einem Doppelklick geöffnet werden. Welche Parameter können verwendet werden, um eine Nachricht zu öffnen?
Produkt David.InfoCenter PC
Priorität normal
Effekt normal
Datum 13.02.2009

Antwort
Um Nachrichten in der Branchensoftware zu öffnen, können verschiedeneAufrufparameter für das InfoCenter (DVWIN32.EXE) genutzt werden. Mit Hilfe dieser Parameter können zum Beispiel eMails, der Kalender-Dialog oder der Kalender Wizard direkt aufgerufen werden.

Folgende Parameter stehen zur Verfügung:

ParameterBedeutung
<file>voll - qualifizierter Dateiname
<path> lokaler Pfadname
[PATH] Pfadname in Servernotation
<n> Dezimalzahl
<x> Hexadezimalzahl

Genereller Aufbau für das Aufrufen von Einträgen aus dem Tobit Archive System:
Syntax: DVWIN32.EXE [PATH] /POS=n /SA=n (+ weitere Modifier s.u.)

  • POS = Position in Archive.DAT
  • SA = Typ (9 = Job, 34 = Archive)

    Optionen:
    Syntax: DVWIN32.EXE /option

    OptionFunktionBeispiel
    /EML file EML Datei öffnen und darstellenDVWIN32 /EML C:\TEMP\PRICELIST.EML
    /ICS file ICS Datei öffnen und darstellenDVWIN32 /ICS C:\TEMP\MEETING.ICS
    /VCF file VCF Datei öffnen und darstellenDVWIN32 /VCF C:\TEMP\RSANDERS.VCF
    /DFML file DFML Dokument öffnen und darstellenDVWIN32 /DFML \\Servername\david\archive\resource\forms\0\id1a9e7c.001
    /001 file 001 Datei öffnen und darstellen (ohne Betreff, An etc.)DVWIN32 /001 C:\TEMP\MESSAGE.001
    /IMG path nImage Messaging Drucker, Pfad und laufende Bild NummerDVWIN32 /IMG C:\TEMP\ 3
    Die Grafiken IMG-001.JPGIMG-002.JPG IMG-003.JPG sollten im Verzeichnis C:\TEMP liegen
    /API=n Faxware Drucker, Nummer des Druckjobs
    Ruft den "Nachrichten versenden" Dialog auf und verwendet die angegebene JobDatei    		DVWIN32 /API=123456789
    JobDatei "0195418B.JOB" und PCX-Grafik "0195418B.001" im Verzeichnis »DAVID\APPS\FAXWARE\OUT\API«. 0195418B ist hierbei der Hexadezimal Wert vomAufrufparameter "26558859".
    /DOC Neues Dokument (ohne cc/bcc)
    /CAL=x:y Neuer Kalendereintrag mit vorgeschlagenem Termin x,y sind hierbei Start- und Stoppzeit in Millisekunden (Hexadezimalwert) seit dem 1.1.1970
    /ADDRESS Addressdialog
    /HWND Fenster der aufrufenden Applikation
    /W Kalender Wizard
    /FROM=file Template für neue Nachricht
    /VCARD User VCard darstellen
    /Options Settings-Dialog darstellen
    /M Modal zu HWND
    -A öffnen eines Archives -A\\Server\david\archive\user\10004000\in

    Modifier:

    /F Forward Mail
    /R Reply Mail
    /A Reply to All
    /D Drag&Drop in Kalender Archive
    /I Received Status der Mail ignorieren (In Bearbeitenmodus öffnen)
    /X Eintrag als Attachment an neue Nachricht anhängen
    /Y In Kalender speichern
    /S Dialog »Erneut Versenden« für diesen Eintrag

    Beispiel:
    Forwarden eines Eintrages:
    DVWIN32.EXE \\SERVER\DAVID\ARCHIVE\USER\10004000\IN /POS=103 /SA=34 /F
