Q-106.236
»Variablen« im Nachrichteneditor


Frage
Problem Wenn im Tobit InfoCenter eine neue Nachricht erstellt oder eine empfangene Nachricht beantwortet beziehungsweise weitergeleitet wird, steht in der Symbolleiste des Nachrichteneditors der Button »Variablen« zur Verfügung.

Wie und wofür kann dieser Button verwendet werden?
Produkt David.InfoCenter PC
Priorität keine
Effekt harmlos
Datum 26.09.2003

Antwort
Auf dem DvISE Server sind viele Information gespeichert, welche bei der Erstellung neuer Nachrichten oft von Hand in den Nachrichtentext geschrieben werden.
Dazu gehören die eigenen Benutzerdaten, Informationen zum Nachrichtenempfänger sowie Datumsinformationen.
Mit dem Button »Variablen« können Sie die auf dem Server gespeicherten Information für Erstellung neuer Nachrichten oder auch eigener Vorlagen zugänglichmachen.

Im folgenden finden Sie eine Auflistung der im Tobit InfoCenter verfügbaren »Variablen« sowie eine Erleuterung der jeweiligen Möglichkeiten.

Benutzerdaten des Absenders

Die Benutzerdaten des Absenders werden den Eingaben im DvISE Administrator unter »Konfiguration -> Benutzer« dem Register Benutzerdaten entnommen.
Dabei wird auf das Benutzerkonto des jeweils angemeldeten Benutzers zugegriffen.

Name:%(Benutzer Name)
Kürzel:%(Benutzer Kürzel)
Titel:%(Benutzer Titel)
eMail:%(Benutzer eMail)
Tel.:%(Benutzer Tel.)
SMS:%(Benutzer SMS)
Zusatz 1:%(Benutzer Zusatz 1)
Zusatz 2:%(Benutzer Zusatz 2)
Zusatz 3:%(Benutzer Zusatz 3)
Zusatz 4:%(Benutzer Zusatz 4)

Daten aus der Adresskarte des Empfängers

Die Informationen zum Nachrichtenempfängers werden den im Tobit InfoCenter unter Adressen angelegten Adresskarten entnommen. Wenn eine Nachricht an mehrere Empfänger gesendet wird, werden die ausgelesenen Adressdaten der eMail des jeweiligen Empängers zugeordnet.

Informationen zum Datum

Mit der Variable »Datum« können Sie das aktuelle Datum in die Nachricht einfügen.
Dabei kann unter verschiedenen Darstellungsweisen gewählt werden.

Datum:%(Datum)
Datum (Tag, Date):%(Datum %A, %x)
Datum (T, Date):%(Datum %a, %x)
Datum (D-M-Y):%(Datum %d-%b-%Y)
Uhrzeit:%(Zeit)
Wochentag:%(Wochentag)
Monat:%(Monat)

Allgemein

Laufende Nummer:%(Laufende Nummer)

Die Variable »Laufende Nummer« nummeriert die einzelnen Nachrichten der Reihe nach durch, beginnend mit 100 in Einserschritten (100, 101, 102, usw.).
Bei jedem Neustart es DvISE Service Layers wird die Zählung mit der nächsten Hunderterstelle fortgesetzt.

Server:%(Server)

Durch das Einsetzten der Variable »Server« wird an der jeweiligen Textstelle der Servername eingetragen.

Benutzerdefiniert:%(Frage "xxxx")

Wenn Sie die Variable »Benutzerdefiniert« auswählen, öffnet sich ein Dialogfenster, in das Sie ein Stichwort eintragen können. Wenn nach der Erstellung der Nachricht der »Senden« Button gedrückt wird, erhalten Sie ein PopUp, welches zur Eingabe des Variablenwertes für diese benutzerdefinierte Variable auffordert. Dieser Wert wird dann an die Textstellen gesetzt, an die Sie diese Variable gestellt haben.

Benutzerdefiniert (einzeln):%(Nachfrage "yyyy"

Diese Variable arbeitet ähnlich wie die Variable »Benutzerdefiniert«, jedoch wird der Variablenwert hierbei für jeden Empfänger abgefragt.
