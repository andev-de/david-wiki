Q-107.373
Auslagern eines David TV TLDs


Frage
Problem Für Aufzeichnungen wird als Kommunikations-Hardware das tvBoard Sat genutzt. Im DvISE Administrator wurde eine WDM Capture Port erstellt. Kann dieser WDM Capture Port für die Aufzeichnungen von TV Sendungen auch auf einen anderen PC auslagert werden?
Produkt TLD (Videoaufzeichnung)
Priorität normal
Effekt normal
Datum 4.01.2005

Antwort
Beim Betrieb von DvISE unter Windows XP oder Windows 2003 kann der WDM Capture Port (TV TLD) ausgelagert werden. Die TV-Karte muss also nicht auf dem DvISE-Hauptserver, sondern kann auf einem beliebigen anderen Windows XP oder Windows 2003 Rechner installiert sein. DvISE ist in der Lage,mit diesem Rechner zu kommunizieren. Diese Installation wird als Remote-Port bezeichnet.

Bevor der WDM-Capture Port auf einen anderen Computer mit Windows XP oder Windows 2003 ausgelagert wird, muss die zu verwendende Hardware (tvBoard Sat) betriebsfertig installiert und funktionstüchtig im DvISE Server eingerichtet sein. Erstellen Sie anschließend im DvISE Administrator einen WDM Capture Port.

Zur Einrichtung des Ports wechseln Sie im DvISE Administrator in den Hauptmenüpunkt »Konfiguration -> Ports« und klicken dann auf das Symbol »Neu«. Daraufhin wird das CSS-Port-Setup gestartet. Klicken Sie hier auf »Hinzufügen«, um einen neuen Port einzurichten oder auf »Überschreiben«, wenn Sie einen bereits bestehenden Port ersetzen wollen. Bei der »Auswahl der Hardware« wählen Sie »WDM Capture Port«. Nachdem Sie alle Punkte des Port Setup durchgegangen sind, wird der erstellte Port im DvISE Administrator unter »Konfiguration -> Ports« angezeigt.

Um die Funktionsfähigkeit der Hardware zu prüfen, sollten Sie eine Aufnahme im David InfoCenter einstellen. Wechseln Sie dazu in das David InfoCenter. Um im David InfoCenter eine Sendung aufzuzeichnen, wechseln Sie in das Archive »TV-Programmübersicht«. Die TV-Programmübersicht ist Ihre Fernsehprogrammzeitschrift und bietet auf einen Blick alle Sendung der ausgewählten Kanäle. Die Einträge der Programmzeitschrift werden nur angezeigt, wenn das Modul TVIndex gestartet ist und ausreichend Guthaben auf Ihrem persönliches Club Tobit Software Konto zur Verfügung stehen. Aufgezeichnete Sendungen werden im Archive »Archive -> Multimedia -> Video -> Aufzeichnungen« abgelegt.

    Hinweise:
    Das Modul TVIndex sorgt für die Erstellung einer digitalen Fernseh-Programmzeitschrift. Im DvISE Administrator können Sie unter dem Hauptmenüpunkt TVIndex die notwendigen Einstellungen zur Konfiguration dieses Moduls vornehmen.

    Die Nutzung des TVIndex ist kostenpflichtig und erfordert eine Anmeldung bei Tobit Software. Die Abrechnung der Kosten erfolgt über den Club Tobit Software, dem Kundenforum von Tobit Software. Weitere Informationen zur Abrechnung und Einblick in Ihr Kundenkonto erhalten Sie im Club Tobit Software unter club.tobit.com.

Bevor jetzt der Remote TLD eingerichtet wird, öffnen Sie im DvISE Administrator die Windows Dienste. In den Eigenschaften des WDM Capture Ports (z.B. DvISE TLD 001) muss die Startart von »Automatisch« auf »Manuell« umgestellt werden. Übernehmen Sie die Änderung und beenden Sie den Dienst. Schließen Sie alle Anwendungen, fahren Sie den PC runter und entfernen Sie das tvBoard aus dem DvISE Server. Starten Sie den DvISE Server nach dem Ausbau des tvBoard Sat neu.

Auf dem Windows XP bzw. Windows 2003 Rechner, auf dem der Remote Port erstellt werden soll, installieren Sie nun das tvBoard Sat. Prüfen Sie in der Windows Systemsteuerung, ob die Hardware betriebsfertig installiert und funktionstüchtig ist.

Auf dem Remote Rechner müssen folgende Änderungen vorgenommen werden:

  • Öffnen Sie den Windows Explorer
  • Erstellen Sie ein Root-Verzeichnis (\DAVID) und geben Sie dieses Verzeichnis frei für »Jeder«.
  • Erstellen Sie die Unterverzeichnisse \DAVID\CODE und \DAVID\TLD\CODE\TV.
  • Kopieren Sie vom DvISE Host-Server die Datei \DAVID\CODE\DAVID.DAT in das Verzeichnis\DAVID\CODE auf den Remote-Server.
  • Kopieren Sie vom DvISE Host Server aus dem Verzeichnis DAVID\TLD\CODE\TV die Dateien »CONFIG.EXE« und »TLD.EXE« in das entsprechenden Verzeichnisse des Remote Servers.
  • Kopieren Sie die Datei DAVIDLIB.DLL und ggf. auch die Dateien »TOBIT32.DLL« und »tgraphic.dll« in das Windows-Systemverzeichnis\SYSTEM32 des Remote Servers.
  • Führen Sie auf dem Remote-Server ein Mapping auf das DvISE-Verzeichnis des Host-Servers durch. Achten Sie darauf, dass für dieses Mapping Administratorrechte erforderlich sind.
  • Dann können den Ports auf dem Remote Rechner mit folgender Syntax starten:LokalerDvISEPfad\TLD.EXE PortNummer PATH=\\Servername\DAVID
Nur wenn der Remote Port fehlerfrei gestartet wurde und somit die Kommunikation zwischen DvISE Host Server und Remote Rechner hergestellt ist, können Aufnahmen erstellt werden.
