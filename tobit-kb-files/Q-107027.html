Q-107.027
Anbindung eines HP LaserJet MFP 4100 an ein DvISE System


Frage
Problem Welche Schritte sind notwendig, um einen HP LaserJet MFP 4100 in Verbindung mit einem DvISE System nutzen zu können?
Produkt Hardware
Priorität keine
Effekt normal
Datum 19.04.2004

Antwort
Neben einer bestehenden David oder FaxWare Installation muss zusätzlich noch FaxGate auf dem DvISE Server installiert werden. Anschließend führen Sie die HP 4100 MFP Hardware Installation durch.

Eine Schritt für Schritt Anleitung steht Ihnen auf der Tobit Webseite unter »Service -> Support -> Getestete Hardware -> Digital Sender« zur Verfügung.
