Q-104.001
Server Locator Services (SLS) mit DSL-Router nutzen


Frage
Problem Was ist bei der Einrichtung des »Server Locator Services« (SLS) zu beachten wenn die aktuelle WAN IP-Adresse des DSL-Router automatisch ausgelesen und an den Tobit DNS-Server übergeben werden soll?
Produkt Server Locator Services
Priorität keine
Effekt harmlos
Datum 27.10.2006

Antwort
Die Server Locator Services for DvISE (SLS) ermöglichen die ständige Online-Präsenz sowie den direkten Zugriff auf Ihren DvISE Server, auch wenn dieser nicht über eine Standleitung mit dem Internet verbunden ist und somit nicht über eine statische IP-Adresse verfügt. Da Computer im Internet ausschließlich über ihre IP-Adresse identifiziert werden, ist der direkte Zugriff auf die Informationen eines Rechners mit wechselnder IP-Adresse praktisch unmöglich.

Eine Änderung der IP-Adresse wird von den SLS automatisch erkannt und dem DNS-Server bei Tobit Software mitgeteilt. Dieser von Tobit Software zentral betriebene DNS Server verwaltet die DNS Einträge sämtlicher Nutzer der Server Locator Services. Erfolgt eine Anfrage über einen Web-oder WAP-Browser, wird diesem vom DNS-Server stets die aktuelle IP-Adresse Ihres DvISE Servers übermittelt. Auf diese Weise ist der DvISE Server über das Internet ständig erreichbar und verhält sich praktisch so, als wäre die Internetanbindung über eine Standleitung realisiert.

Sollen auch die Möglichkeiten des Fernzugriffs auf die eigenen Nachrichten genutzt werden, muss zusätzlich das Internet Access Module for DvISE eingesetzt werden, das den Web- und WAP-Server beinhaltet. Weiterhin muss eine »Always-Online-Verbindung« zum Internet bestehen, z.B. über eine Flatrate (wie DSL Flat). Für die Nutzung der SLS muss außerdem der DvISE PostMan auf dem DvISE Server eingerichtet sein. Aus diesem Grund wird der DvISE PostMan bei der SLS Installation automatisch gestartet, auch wenn dieser Dienst bisher nicht genutzt wurde.

Die Installation der Server Locator Services for DvISE geschieht in der Regel direkt bei der DvISE-Grundinstallation. Aktivieren Sie die Option »Server Locator Services for DvISE installieren«. Im folgenden Dialogfenster werden Sie zur Eingabe eines Kennwortes aufgefordert. Tragen Sie hier den dritten Zeichenblock Ihrer Systemlizenz ein. Diese Zeichenkette ist gleichzeitig Ihr Kennwort für das Kundenforum Club Tobit unter club.tobit.com. Falls Sie sich bereits im Club Tobit registriert haben und eventuell auch schon eine Passwortänderung vorgenommen haben, so tragen Sie an dieser Stelle das aktuell gültige Passwort ein. Dieses Passwort dient zur Authentifizierung für die regelmäßige Übermittlung der IP-Adresse an den DNS-Server von Tobit Software.

Ihre Zugangsdaten werden automatisch generiert und im Tobit Archive System auf Ihrem DvISE Server unterhalb des Archives »System -> DvISE -> Domänen -> Authentifizierungs Domänen« gespeichert. Ihr aktueller Domain Name setzt sich aus Ihrer Site-ID (die ersten 10 Stellen Ihrer Systemlizenz inkl. Bindestrich) und der Domain »tobit.net« zusammen, also z.B.:

12345-67890.tobit.net

Dieser Domain Name wird in der PostMan Konfiguration unter »Datenbanken -> Gültige Domain Namen« eingetragen.

Im Club Tobit unter club.tobit.com können Sie zusätzlich zu diesem automatisch vergebenen Domain Namen weitere frei wählbare Subdomains von tobit.net für sich einrichten, also z.B. »mustermann.tobit.net«, sofern diese nicht bereits anderweitig vergeben wurden. Auch die Verwendung eigener Hauptdomänen (Second Level Domains), wie z.B. »example.com« ist möglich. Beachten Sie hierbei, dass diese Domain Namen dann ebenfalls in der DvISE Konfiguration (z.B. bei WebBox und PostMan) eingetragen werden müssen.

Im DvISE-Administrator unter »Postman-> Erweitert-> Routerkonfiguration« müssen Sie dann Ihren DSL-Router auswählen (nur möglich wenn Sie den Haken bei »Service Locator Service« und »Benutzung eines Routers« setzen) und alle notwendigen Daten hinterlegen.
Tragen Sie dann die IP-Adresse ihres Routers, sowie den TCP Port über den Sie den Router über den Web-Browser konfigurieren können, den Benutzernamen und das Passwort ein. Über den Button »Testen« können Sie überprüfen ob die aktuelle WAN IP-Adresse ausgelesen werden kann.Übernehmen Sie die Einstellungen unter starten Sie unter »DvISE-Dienste« den "DvISE-Postman" neu.

Jetzt wird die aktuelle WAN IP-Adresse automatisch ausgelesen und an die Service Locator Services übergeben.
