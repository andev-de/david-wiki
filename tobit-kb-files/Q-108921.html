Q-108.921
Hinweise zur Nutzung des von AGFEO angepassten »CONNECT.DCC« Scripts


Frage
Problem David wird in Verbindung mit einer AGFEO TK-Anlage eingesetzt. Hierbei soll nun auch das von AGFEO angepasste Script »CONNECT.DCC« verwendet werden. Welche Schritte sind dazu notwendig und was ist zu beachten?
Produkt TLD (allgemein)
Priorität normal
Effekt normal
Datum 11.12.2009

Antwort
Das Script »CONNECT.DCC« wird u.a. benötigt, damit David eine ausgehende Telefon-Verbindung durch Nutzung von CTI Funktionen (Computer Telephony Integration) über eine TK-Anlage aufbauen kann. Diese Datei befindet sich auf Ihrem David Server im Verzeichnis »\DAVID\TLD\COMMON\«. Nähere Informationen dazu erhalten Sie im KnowledgeBase Artikel Q-106.271.

Der TK-Anlagen-Hersteller AGFEO hat für Nutzer von AGFEO TK-Anlagen ein spezielles »CONNECT.DCC« Script erstellt, welches dem David Benutzer zusätzliche Funktionen zur Verfügung stellt:

  • Direktes Starten des Systemtelefons im Freisprech-Betrieb bei Rufnummernwahl mit David.
  • Direktes Starten des Systemtelefons im Headset-Betrieb bei Rufnummernwahl mit David.
    Verfügbar, wenn ein Headset an Ihr AGFEO Systemtelefon angeschlossen ist. Unterstützte AGFEO Systemtelefone:
    • »ST 21«
    • »ST 25«
    • »ST 30«
    • »ST 40«
  • Einschränkungen:
    • Die Wiedergabe von Voicemails auf Ihrem Systemtelefon ist bei Nutzung des Freisprech- oder Headset-Betriebs nicht mehr möglich.
    • Externe Teilnehmer mit dreistelliger Rufnummer können bei Nutzung des AGFEO »CONNCECT.DCC« Scripts nicht angewählt werden, da diese Rufnummern im Script als intern ausgewertet werden.
    • In Ihren David Adresskarten dürfen sich keine »!« vor den Rufnummern befinden, da diese vom AGFEO »CONNCECT.DCC« Script nicht richtig ausgewertet werden.
      (Im »CONNECT.DCC« Script von Tobit.Software würde erst der externe Teilnehmer angerufen werden)
    • Bei Teilnehmern mit analogen Nebenstellen muss in den Einstellungen des David Client im Bereich »Rufnummer für TAPI-Funktionen (Lokale Nebenstelle)« das Präfix »a« vor die Rufnummer gesetzt werden. Ohne das Präfix »a« erhält der Anwender bei Nutzung der Wahlhilfe im David Client eine Meldung mit dem Hinweis auf einen unbekannten Fehler, während das Telefongespräch zu der gewünschten Zielnummer zwar aufgebaut aber ungewollt direkt mit dem Anrufbeantworter des David Benutzers verbunden wird.

Hinweis: Die zum Download bereitgestellte Datei liegt im »ZIP« Format vor. Zum Öffnen wird ggf. ein Packprogramm benötigt.

Folgende Voraussetzungen müssen für den Einsatz des angepassten »CONNECT.DCC« Scripts erfüllt sein:

  • Eingerichteter David Server (David 6.6 oder höher), insbesondere:
    • Eine im Server installierte ISDN-Karte
      (siehe Liste mit unterstützter Hardware))
    • Ein in David eingerichteter ISDN-Port (TLD) mit aktiviertem »ECT« Dienst
  • Installiertes AGFEO »CONNECT.DCC« Script
  • Angepasste Einstelllungen im David Client
  • Aktivierte CTI-Funktion in der AGFEO-TK-Anlage

Installation des AGFEO »CONNECT.DCC« Scripts

Gehen Sie folgendermaßen vor, um das »CONNECT.DCC« von AGFEO zu installieren:

Prüfen der eingerichteten ISDN-Ports

  1. Öffnen Sie den David.Administrator und wechseln Sie im Konfigurationsbaum auf der linken Seite in den Bereich »System > Ports«.
  2. Markieren Sie einen ISDN-Port mit der rechten Maustaste und wählen Sie aus dem Kontextmenü den Eintrag »Eigenschaften«.
  3. Überprüfen Sie auf der Registerkarte »Dienste«, ob die Option »ECT« aktiv ist. Notieren Sie sich in diesem Fall die Nummer des Ports (z.B. »001«) und schließen das Fenster mit den Port-Eigenschaften.
  4. Wiederholen Sie diesen Vorgang für alle eingerichteten ISDN-Ports.

Durchführen der Installation

  1. Rufen Sie in der Symbolleiste des David.Administrator über die Schaltfläche »David Dienste« das zugehörige Dialogfenster auf und beenden Sie alle David Dienste.

  2. Erstellen Sie auf Ihrem David Server ein temporäres Backup-Verzeichnis außerhalb der David Verzeichnis-Struktur, z.B. »C:\BACKUP-DAVID\«.
  3. Wechseln auf Ihrem David Server mit dem Windows Explorer in das Verzeichnis »\DAVID\TLD\COMMON\«.
  4. Verschieben Sie die dort vorhandene Datei »CONNECT.DCC« in das soeben erzeugte Backup-Verzeichnis, um diese bei Bedarf wiederherstellen zu können.
  5. Kopieren Sie die von AGFEO angepasste Datei »CONNECT.DCC« in das Verzeichnis »\DAVID\TLD\COMMON\« Ihres David Servers.
  6. Wechseln Sie in das Verzeichnis »\DAVID\TLD\PORT\«. Hier befindet sich u.a. für jeden eingerichteten ISDN-Port ein eigenes Unterverzeichnis, wobei dessen Name der jeweiligen Port-Nummer entspricht.

  7. Markieren die Verzeichnisse aller Ports, die Sie zu Anfang notiert haben, und kopieren Sie diese Verzeichnisse ebenfalls in das zuvor erstellte Backup-Verzeichnis.
  8. Wechseln Sie nun nacheinander in die Original-Verzeichnisse aller von Ihnen notierten Ports, z.B. »\DAVID\TLD\PORT\001\«.
  9. Öffnen Sie die darin enthaltene Datei »TLD.INI« mit einem Editor und fügen Sie in einer neuen Zeile den folgenden Parameter ein:
    VALIDNUMBERSIGNS=*#

    10. Speichern Sie Ihre Änderungen an der Datei ab und wiederholen Sie diesen Vorgang bei allen »TLD.INI« Dateien, bei deren Ports die »ECT« Option aktiv ist.

  10. Nachdem Sie alle betreffenden »TLD.INI« Dateien angepasst haben, öffnen Sie den David.Administrator, rufen in der Symbolleiste über die Schaltfläche »David Dienste« das zugehörige Dialogfenster auf und starten Sie die zuvor beendeten David Dienste wieder.

Anpassung der Einstellungen im David Client

Sie haben die Möglichkeit, zu bestimmen, ob Ihr Systemtelefon bei Nutzung der Rufnummernwahl mit David generell im Freisprech- oder im Headset-Betrieb starten soll. Die hierzu erforderlichen Einstellungen können Sie direkt im David Client vornehmen. Gehen Sie dazu folgendermaßen vor:

  1. Öffnen Sie den David Client und wechseln Sie im Hauptmenü in den Bereich »Optionen > Einstellungen«.
  2. Fügen Sie im Bereich »Benutzerprofil« in das Feld »Rufnummer für TAPI-Funktionen (Lokale Nebenstelle)« in Abhängigkeit von der gewünschten Funktion folgendes ein:
    • Für den generellen Start im Headset-Betrieb
      Tragen Sie die interne Rufnummer Ihrer lokalen Nebenstelle ein, angeführt von einem »h«, z.B. »h16« für Headset-Betrieb an der internen AGFEO Nebenstelle 16.
    • Für den generellen Start im Freisprech-Betrieb
      Tragen Sie die interne Rufnummer Ihrer lokalen Nebenstelle ein, angeführt von einem »a«, z.B. »a16« für Freisprech-Betrieb an der internen AGFEO Nebenstelle 16.

  3. Speichern Sie Ihre Einstellungen durch einen Klick auf »OK« ab.

Hinweise zur Aktivierung der CTI-Funktion in der AGFEO-TK-Anlage

Um die CTI Funktion mit der AGFEO TK-Anlage nutzen zu können, muss für die gewünschte Nebenstelle die Fremdwahl mit CTI in der AGFEO TK-Anlage aktiviert werden. Gehen Sie dazu folgendermaßen vor:

  1. Lesen Sie mit dem AGFEO Programm »TK Suite« die Konfiguration Ihrer TK-Anlage aus.
  2. Aktivieren Sie bei der gewünschten Nebenstelle die Fremdwahl mit CTI im Bereich »Telefoneinstellungen > Wählen«.

Technische Unterstützung zur Einrichtung Ihrer TK-Anlage erhalten Sie beim AGFEO Support oder unter www.agfeo.de.
