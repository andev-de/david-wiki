Q-102.789
Fehlermeldung bei dem Druck mit dem Fax-Image Drucker


Frage
Problem Wenn versucht wird, über den Fax-Image Drucker ein Dokument auszudrucken, wird die folgende Meldung ausgegeben:

Windows kann wegen eines Problems mit der Druckereinrichtung nicht drucken.
Testen sie die folgenden Optionen.
*Drucken sie eine Testseite aus Windows
*Überprüfen Sie, ob der Drucker eingeschaltet ist
*Installieren Sie den Druckertreiber neu.

Das Problem betrifft nicht alle Workstations.
Produkt David
Priorität dringend
Effekt kritisch
Datum 13.01.2003

Antwort
Abhängig von der verwendeten MAPI32.DLL im Verzeichnis »WINNT\SYSTEM32« (Windows NT/W2K/XP) wird nach erfolgreicher Renderung des Dokumentes zum PCX File dieses an den Standard eMail Client übergeben.

Der erste Schritt zur Problemlösung stellt die saubere De-/Neuinstallation des Fax-Image Druckers dar. Hierbei wird stets davon ausgegangen, dass Sie als Administrator angemeldet sind.

Vorgehensweise bei Verwendung des
Connector for MS Exchange for DvISE

  1. Deinstallieren Sie von einem betroffenen Client aus, aus dem Serververzeichnis »PROGRAMME\DAVID MX (bzw. DvISE Connector for Exchange) \OUTLOOK\FAXIMG\WINNT« mittels SETUP.EXE, zunächst den Fax-Image Drucker ohne ihn neu zu installieren.
    Rebooten Sie die Workstation anschließend.
  2. Führen Sie nun erneut die SETUP.EXE aus und installieren Sie den Fax-Image Drucker ohne ihn zu deinstallieren.
    Rebooten Sie das System erneut.
  3. Prüfen Sie die Funktionalität mit einem testweise erzeugten MS Word Dokument.

Vorgehensweise bei Verwendung unter David 6.x/XL

  1. Deinstallieren Sie von einem betroffenen Client aus, aus dem Serververzeichnis »DAVID\CLIENTS\WINDOWS\FAXIMG« mittels SETUP.EXE, zunächst den Fax-Image Drucker ohne ihn neu zu installieren.
    Rebooten Sie die Workstation anschließend.
  2. Führen Sie nun erneut die SETUP.EXE aus und installieren Sie den Fax-Image Drucker ohne ihn zu deinstallieren.
    Rebooten Sie das System erneut.
  3. Prüfen Sie die Funktionalität mit einem testweise erzeugten MS Word Dokument.
Weitergehende Maßnahmen
Sollte durch die De-/Neuinstallation des Fax-Image Druckertreibers noch keine Funktionalität gegeben sein, so prüfen Sie die folgenden Punkte:
  1. Befinden sich nach der Installation die folgenden Dateien in den angegebenen Verzeichnissen. Vergleichen Sie die Versions- und Größenangaben mit den Dateien aus dem Installationsverzeichnis:
    • DVMAILER.EXE,
      Verzeichnis »WINNT\«
    • FWRASDD.DLL,
      Verzeichnis »WINNT\system32\spool\drivers\w32x86«
    • FWIMGMO.DLL,
      Verzeichnis »WINNT\system32«
    • FAXIMG.DLL,
      Verzeichnis »WINNT\system32\spool\drivers\w32x86«

  2. Prüfen Sie in den Eigenschaften des Druckertreiber die Anschlüsse:
    Als Command muss der Pfad zur DVMAILER.EXE definiert sein:
    C:\WINNT\DVMAILER
    Als Directory muss das Faximg-Verzeichnis (z.B. »C:\FaxImg\«) hinterlegt sein.

  3. Die in der Problembeschreibung genannte Fehlermeldung kann auch erzeugt werden, wenn Sie umfangreiche und verschachtelte Exceltabellen rendern wollen.
    Testen Sie daher die Grundfunktionalität stets mit MS Word und einem eigens dafür angelegtem Worddokument mit einigen Testzeilen.
