Q-109.722
Konfiguration und Nutzung von TV Select in Verbindung mit David.fx


Frage
Problem TV Select soll zusammen mit David.fx genutzt werden. Welche Möglichkeiten ergeben sich dadurch, und welche Einstellungen sind dafür notwendig?
Produkt TV Select
Priorität normal
Effekt normal
Datum 15.10.2010

Antwort
TV Select ist eine kostenfreie App für das iPhone und den iPod touch und zeigt das aktuelle Fernsehprogramm der TV DIGITAL, Deutschlands populärstem TV-Magazin, an. In Verbindung mit David.fx kann TV Select auch als Instrument für die Aufnahmeplanung von TV-Sendungen und als Fernbedienung zur Steuerung des David Mediaplayers genutzt werden:
  • Grundeinrichtung
  • Vornehmen von Aufzeichnungen und Aufnahmeplanung in TV Select
  • Fernsteuerung des David Mediaplayer

Grundeinrichtung

Um TV Select in Verbindung mit David.fx nutzen zu können, muss zwischen TV Select und dem David.fx Server eine Verbindung aufgebaut werden.

Dazu können Sie beim iPhone folgende Möglichkeiten nutzen:

  • Verbindungsaufbau per EDGE / UMTS,
    wenn der David.fx Server von extern über das Internet erreichbar ist. Nähere Informationen dazu finden Sie im Abschnitt zur externen Nutzung.
  • Verbindungsaufbau per WLAN

Beim iPod touch besteht ausschließlich die Möglichkeit, eine Verbindung per WLAN zum David.fx Server aufzubauen. Netzwerkintern sollten Sie darauf achten, Ihren Geräten (PCs, Server, iPhones) statische IP-Adressen zuzuweisen. DHCP-Adressenvergabe sollten Sie dann z.B. in Ihrem Router deaktivieren.

Auf dem David.fx Server muss das Programm »David Mediaplayer« dauerhaft gestartet sein. Dieses Programm befindet sich auf Windows-Verzeichnisebene als »DVWINTV.EXE« im Verzeichnis »C:\PROGRAMME\TOBIT INFOCENTER«. In der Konfiguration des David Mediaplayer muss unter »Datei > Einstellungen > Remote-Control« die Option »Remote-Control erlauben« aktiviert werden. Das Anlegen von Benutzername und Passwort ist optional, wird aus Sicherheitsgründen aber empfohlen.

Starten Sie anschließend den David Mediaplayer neu. Achten Sie darauf, dass im Windows Task-Manager unter »Prozesse« nur einmal der Prozess »DVWINTV.EXE« aufgeführt ist. Existiert hier mehr als ein Eintrag mit diesem Namen, beenden Sie alle DVWINTV-Prozesse und starten Sie anschließend den David Mediaplayer neu. Netzwerkintern ist nun eine Verbindung zum David.fx Server möglich. Falls eine Firewall genutzt wird oder eine Verbindung von extern über das Internet aufgebaut werden soll, beachten Sie folgende Hinweise:

  • Der TCP-Port 8899 muss für eingehende Verbindungen in Ihrem Router und Ihrer Firewall geöffnet sein.
  • Ihr David.fx Server muss von extern über das Internet erreichbar sein. Z.B. durch Einsatz einer statischen IP-Adresse, die Sie von Ihrem Internet-Provider bekommen haben, oder durch Nutzung der »Server Locator Services«, den Sie als kostenpflichtigen Zusatzdienst im David.Administrator innerhalb der PostMan Konfiguration auf der Registerkarte »Generell« aktivieren können. Beachten Sie dazu den KnowledgeBase Artikel Q-100.229.

In TV Select müssen Sie anschließend im Bereich »Mehr > Server« den David.fx Server angeben, mit dem Sie sich verbinden möchten. Klicken Sie dazu oben rechts auf die Schaltfläche »Bearbeiten« und anschließend oben links auf die »+« Schaltfläche. TV Select wird Ihnen daraufhin alle aktuell im Netzwerk verfügbaren David.fx Server anzeigen, bei denen der David Mediaplayer mit aktivierter Remote-Control gestartet ist.

Nach Auswahl Ihres David.fx Servers klicken Sie erneut auf den Server-Namen bzw. auf das blaue Pfeil-Symbol dahinter um ggf. die Zugangsdaten einzutragen, die Sie zuvor in den Remote-Control-Einstellungen des David Mediaplayer definiert haben (Benutzername und Passwort). Wechseln Sie anschließend über die Schaltfläche »Server« zurück zur Server-Übersicht und bestätigen Sie Ihre Einstellungen mit einem Klick auf die Schaltfläche »Fertig«.

Nach erneutem Anklicken des Server-Namens wird eine Verbindung zum David.fx Server aufgebaut. In der Server-Übersicht wird eine aktive Verbindung hinter der IP-Adresse oder dem Server-Namen mit einem Häkchen angezeigt.

Alternativ können Sie die Zugangsdaten zum David.fx Server auch manuell in der Server-Übersicht über den Menüpunkt »Manuelle Eingabe« hinterlegen. Dabei stehen Ihnen folgende Optionen zur Verfügung:

  • IP-Adresse, unter der der David.fx Server von intern oder extern erreichbar ist, gefolgt von einem Doppelpunkt und dem für die Verbindung zu verwendenden TCP-Port (8899)
    Beispiel: 192.168.0.1:8899
  • Domain Name, unter der der David.fx Server von intern oder extern erreichbar ist, gefolgt von einem Doppelpunkt und dem für die Verbindung zu verwendenden TCP-Port (8899)
    Beispiel: example.net:8899
  • Rechner-Name des David.fx Servers

Eine bestehende Verbindung zwischen TV Select und einem David.fx Server können Sie unter »Mehr > Server« löschen, indem Sie in den »Bearbeiten«-Modus wechseln, den betreffenden Server auswählen und auf die Schaltfläche »Server entfernen« klicken.

Vornehmen von Aufzeichnungen und Aufnahmeplanung in TV Select

Beim Aufrufen von Detail-Informationen zu einer Sendung, steht in TV Select oben rechts die Schaltfläche »Aufnehmen« zur Verfügung. Um diese Funktion nutzen zu können, muss David.fx gemäß dem KnowledgeBase Artikel Q-109.955 zum Aufzeichnen von TV-Sendungen eingerichtet sein.

Ist dies der Fall, können Sie durch einen Klick auf die »Aufnehmen« Schaltfläche und das Bestätigen der Aufzeichnungsabfrage mit »Ja« einen neuen Aufnahme-Auftrag in David generieren. Dieser ist im David Client unter »Navigator > Programm« in der Spalte »Aufnahmen« sichtbar und kann dort bei Bedarf auch wieder gelöscht werden. Die Aufzeichnungen werden im David Client unter dem Ordner »Multimedia > Video > Aufzeichnungen« gespeichert.

Fernsteuerung des David Mediaplayer

Wenn Sie in TV Select den Bereich »Mediathek« aufrufen, wird Ihnen eine Eintragsliste mit allen Aufnahmen angezeigt, die auch im David Client unter dem Ordner »Multimedia > Video > Aufzeichnungen« sichtbar sind.

Nach Auswahl einer Sendung befindet sich über den Detail-Informationen zur Aufzeichnung oben rechts die Schaltfläche »Wiedergabe«. Klicken Sie die Schaltfläche an, um den David Mediaplayer die Aufzeichnung auf Ihrem PC bzw. Server wiedergeben zu lassen.

In TV Select werden nun eine Zeitleiste sowie verschiedene Schaltflächen zur weiteren Steuerung der Wiedergabe angezeigt:

  • Halten Sie den roten Pfeil auf der Timeline gedrückt und bewegen Sie den Pfeil nach unten oder oben, um in der Aufzeichnung vor- oder zurückzuspulen.
  • Nach Anklicken der oberen rechten Schaltfläche erscheint ein Schieberegler, über den Sie die Lautstärke anpassen können.
  • Die Schaltflächen im unteren Bereich haben folgende Aufgaben (von links nach rechts): Vorherige Aufzeichnung wiedergeben, zurückspulen, pausieren/wiedergeben, vorspulen und nächste Aufzeichnung wiedergeben.
  • Nach Anklicken der oberen linken Schaltfläche wechseln Sie zur vorherigen Seite mit den Detail-Informationen zur Aufzeichnung zurück. Von dort aus können Sie auch zurück zur Eintragsliste wechseln.

Die Fernsteuerung des David Mediaplayer mit TV Select ist auf allen Computern möglich, die Zugriff auf den David.fx Server besitzen, über einen korrekt installierten David Client verfügen und deren David Mediaplayer für den Remote-Control-Betrieb wie oben beschrieben konfiguriert ist.

Allgemeine Hinweise:
Die Nutzung von TV Select ist sowohl mit der David.fx Basic als auch mit der David.fx Pro Edition möglich.
Die Installation von TV Select erfolgt direkt über den App Store. Entweder direkt auf dem iPhone / iPod touch oder über iTunes.
