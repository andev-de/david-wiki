Q-101.414
Anpassen des InfoCenter Notifier im Tobit InfoCenter XP


Frage
Problem Der InfoCenter Notifier des Tobit InfoCenter XP soll entsprechend der Bedürfnisse der einzelnen Benutzer angepasst werden.
Produkt David.InfoCenter
Priorität keine
Effekt harmlos
Datum 2.01.2007

Antwort
Um den Konfigurationsdialog aufzurufen, klicken Sie mit der rechten Maustastedas Symbol des Notifiers im SysTray rechts unten auf Ihrem Bildschirm nebender Uhrzeit an und wählen aus dem Menü »Konfiguration...«.

Registerkarte "Einstellungen"

  • Info zeigen
    Beim Eintreffen einer neuen Nachricht in einem der überwachten Archives öffnet sich das Infofenster des Notifiers. Es zeigt in einer Liste alle neuen bzw. ungelesenen Nachrichten zusammen jeweils mit dem Betreff der Nachricht unddem Archive, in dem die Nachricht liegt. Durch Doppelklick auf die Betreffzeile wird die Nachricht im passenden Viewer geöffnet. Sollte das Tobit InfoCenter XP zu diesem Zeitpunkt noch nicht geladen sein, so wird es nun gestartet.

  • Einblenden
    Aktivieren Sie diesen Schalter, und das Nachrichtenfenster wird sanft eingeblendet, wenn eine neue Nachricht eintrifft. Diese Option steht nur zur Verfügung, wenn der Schalter »Info zeigen« aktiviert ist.

  • Kurzinfo
    Wenn Sie den Schalter »Kurzinfo« auswählen, wird bei Nachrichteneingang ineins der überwachten Archives ein Hinweisfenster eingeblendet, welches denNamen des betreffenden Archives, die Anzahl der neuen oder ungelesenenNachrichten und Betreffzeile und Absender der letzten eingetroffenen Nachricht anzeigt.Sobald Sie das Hinweisfenster anklicken, wechselt das Tobit InfoCenter XP indas betreffende Archive. Sollte das Tobit InfoCenter XP zu diesem Zeitpunkt noch nicht geladen sein, so wird es nun gestartet. Nach einigen Sekundenschließt sich das Hinweisfenster selbstätig.

  • Sprechblase
    Wenn Sie diesen Schalter einschalten, wird ausgehend vom Notifier-Symbol im Windows Systray eine Sprechblase angezeigt. Diese enthält neben der Anzahl der neuen Nachrichten Betreffzeile und Absender der neuesten Nachricht. Der Schalter ist nur verfügbar, wenn Sie zuvor »Kurzinfo« aktiviert haben.

  • System Beep
    Bei Eingang einer neuen Nachricht wird vom Systemlautsprecher ein Signal gegeben.

  • Sound spielen
    Bei Empfang einer neuen Nachricht wird vom Soundsystem der eingestellteKlang abgespielt. Die Sound-Datei kann in der Microsoft Windows-Systemsteuerungausgewählt werden.

  • Keyboard Flash
    Bei Eingang einer neuen Nachricht blinken die Leuchtdioden rechts oben aufIhrer Tastatur in bestimmten Abständen.

  • TrayIcon
    Wählen Sie hier ein Symbol, das für den Notifier im Windows Systray angezeigtwird. Wenn eine neue Nachricht eintrifft, verändert sich die Symbolgröße in bestimmten Abständen. Das Icon "InfoCenter Notifier XP" zeigt zusätzlich dieAnzahl der neu eingetroffenen Nachrichten an.

  • Info immer im Vordergrund
    Nach Setzen dieses Schalters bleibt das Infofenster (s. »Info zeigen«) nach demErscheinen ständig sichtbar und kann nicht durch andere Fenster überdecktwerden. Dieser Schalter ist allerdings nur dann aktiv, wenn Sie den Schalter»Info zeigen« ausgewählt haben.
Registerkarte "Überwachte Archives"
Das Fenster zeigt Ihnen die Liste der Verzeichnisse, die vom Notifier ständig auf neue Nachrichten überprüft werden. Nach der Installation ist dort nur das Eingangsbuch eingetragen. Sie können jedoch jedes andere Archive überwachenlassen. Das ist z.B. dann sinnvoll, wenn Sie Verteilregeln definiert haben, d.h., wenn eintreffende Nachrichten nach bestimmten Kriterien in verschiedeneArchives verteilt werden.

Wenn Sie ein in der Liste aufgeführtes Archive aus der Überwachung durchden InfoCenter Notifier löschen möchten, markieren Sie das gewünschte Archiveund klicken Sie auf die Schaltfläche »Löschen«.

  • Ausgangsbuch überprüfen
    Bei Auswahl dieses Schalters wird das Ausgangbuch in die Liste der überwachtenArchives übernommen. Dadurch werden Nachrichten, die aus irgendeinem Grund nicht versendet werden konnten und im Status-Feld eine Fehlermeldungenthalten, über den Notifier gemeldet.

  • Fehler-Benachrichtigung
    Mit Aktivierung dieses Schalters wird immer dann, wenn eine Nachricht imAusgangsbuch einen Fehlerstatus erhält, und demzufolge endgültig nicht versendetwerden konnte, eine entsprechende Nachricht ins Eingangsbuch gestellt.Diese Nachricht weist Sie mit dem Text "Bitte prüfen Sie Ihr Ausgangs-Archive!"auf den Fehler hin.
Registerkarte "Erweitert"
  • Scan
    Hier wird eingestellt, in welchen Zeitabständen der Notifier die überwachtenArchives nach neu eingetroffenen Nachrichten untersucht.

  • Beep/Sound
    Legen Sie hier eine Zeitspanne fest, nach deren Ablauf sich eine neu eingetroffene Nachricht mit einem Sound in Erinnerung ruft. Erst nachdem Sie die Nachricht geöffnet haben, verstummt diese Erinnerung. Diese Option ist nur auswählbar, wenn Sie auf der Registerkarte "Einstellungen" (s.o.) den Schalter»Sound spielen« aktiviert haben.
Standardmäßig wird der InfoCenter Notifier automatisch mit dem Tobit InfoCenter XP zusammen gestartet. Diese Funktion können Sie deaktivieren, indem Sie im Tobit-InfoCenter-XP-Startbildschirm unter »Optionen -> Einstellungen -> Registerkarte "Ansicht" den Schalter »InfoCenter Notifier starten« deaktivieren.
