Q-109.955
TV-Aufzeichnungen mit David.fx


Frage
Problem David.fx soll für das Aufzeichnen von TV-Sendungen genutzt werden. Welche Voraussetzungen müssen dafür gegeben sein?
Produkt David
Priorität normal
Effekt normal
Datum 15.10.2010

Antwort
David.fx kann sowohl in der Basic als auch in der Pro Edition für das Aufzeichnen von TV-Sendungen eingesetzt werden. Folgende Voraussetzungen müssen dafür erfüllt sein:
  • Der David Server muss unter Windows XP (SP3), Windows Vista, Windows 7 oder Windows Server 2003 betrieben werden.
  • TV-Karte für DVB-S- oder DVB-T-Empfang. Für diese Karte müssen entweder WDM (Windows Driver Model) oder BDA (Broadcast Driver Architecture) Treiber zur Verfügung stehen. Geeignete TV-Karten finden Sie in der Liste der mit David getesteten Hardware.
  • Installierter MPEG-2-Codec (z.B. durch Installation einer bei der TV-Karte mitgelieferten DVD-Software oder durch Installation eines Codec Packs) auf dem David Server sowie auf den Arbeitsstationen.
  • Aktiviertes Backline Feature »TV Services«. (Nähere Informationen dazu finden Sie weiter unten im Artikel.)

Die Bereitstellung der Aufnahmefunktion richten Sie auf dem David.fx Server im David.Administrator ein. Die Verwaltung der Aufzeichnungen und die Aufnahmeplanung erfolgt generell im David Client:

  • Grundeinrichtung im David.Administrator
  • Vornehmen von Aufzeichnungen und Aufnahmeplanung im David Client

Grundeinrichtung im David.Administrator

Sobald die TV-Karte in Ihrem David.fx Server eingebaut und der Treiber korrekt installiert worden ist, müssen Sie im David.Administrator unter »System > Ports« einen neuen Port über die Schaltfläche »Hinzufügen« installieren.

Wählen Sie im Port Setup den Punkt »WDM Capture Port« aus und klicken Sie auf »Weiter«. Das Setup-Programm führt Sie automatisch durch die Port-Installation. Die Option »Hardware automatisch erkennen« sorgt dafür, dass die eingebaute TV-Karte dabei selbstständig erkannt und eingebunden wird. Falls diese Funktion fehlschlägt oder manuell deaktiviert wurde, erscheint eine Liste, aus der Sie die betreffende Karte manuell auswählen können.

Handelt es sich bei der von Ihnen eingesetzten TV-Karte um ein WDM-Modell, müssen Sie zunächst die bei der TV-Karte mitgelieferte Software starten, um damit einen ersten Sendersuchlauf durchzuführen. Nach Abschuss dieses Suchlaufs wird die Senderliste durch den WDM-Treiber auch für David zur Verfügung gestellt. Rufen Sie anschließend im David.Administrator unter »System > Ports« per Kontextmenü die Eigenschaften Ihres WDM Capture Ports auf und klicken Sie in der Registerkarte »Audio/Video« auf »Suchlauf«, um die Senderliste zu laden.

Bei Verwendung einer TV-Karte, die mit BDA-Treibern arbeitet, ist ein Starten von Software aus dem Lieferumfang der Karte nicht erforderlich. Öffnen Sie im David.Administrator unter »System > Ports« die Eigenschaften des TV-Ports und Starten Sie den Suchlauf durch einen Klick auf die zugehörige Schaltfläche in der Registerkarte »Audio/Video«.

Um nun Aufzeichnungen vornehmen zu können, wird der David Dienst »TVIndex« benötigt. Dieser Dienst ist an das Backline Feature »TV Services« gebunden und liefert Ihnen eine elektronische Programmzeitschrift für die Aufnahmeplanung.

Hinweis:
Die Nutzung der »TV Services« ist kostenpflichtig und erfordert eine Anmeldung bei Tobit.Software. Die Abrechnung der Kosten erfolgt im Tobit.NET, dem Kundenforum von Tobit.Software. Weitere Informationen zur Abrechnung sowie Einblick in Ihr Kundenkonto erhalten Sie direkt im Tobit.NET unter www.tobit.net.

Starten Sie diesen Dienst in dem Fenster »David Dienste« das Sie über die gleichnamige Schaltfläche in der Symbolleiste des David.Administrator öffnen können.

Nachdem der »TVIndex« Dienst gestartet ist, müssen Sie ggf. in der Senderliste Ihres TV-Ports einigen Sendern einen alternativen Sendernamen zuweisen, um sicherzustellen, dass eine eindeutige Zuordnung zwischen den Kanalbezeichnungen der elektronischen Programmzeitschrift und der von der TV-Karte gelieferten Kanalliste möglich ist. Vergleichen Sie also die Bezeichnungen in der Kanalliste mit denen der elektronischen Programmzeitschrift. Stellen Sie Unterschiede fest, tragen Sie den in der Programmzeitschrift verwendeten Namen in der Kanalliste des TV-Ports als alternativen Namen für den betreffenden Kanal ein. Markieren Sie dazu den zugehörigen Eintrag in der Kanalliste des Ports (z.B. »Das Erste«) und klicken Sie auf »Bearbeiten«. Daraufhin können Sie auf der Registerkarte »Allgemein« den alternativen Namen eintragen (z.B. »ARD«).

Im David.Administrator können Sie bei Bedarf die »Dynamische Speicherverwaltung der TV-Aufzeichnungen« unter »System > Kontextmenü > Konfigurieren« in der Registerkarte »Bereinigung« aktivieren, um zu verhindern, dass sämtlicher freier Speicherplatz Ihrer Festplatte durch TV-Aufzeichnungen belegt wird.

Vornehmen von Aufzeichnungen und Aufnahmeplanung im David Client

Starten Sie nach der Grundeinrichtung im David.Administrator Ihren David Client neu. Anschließend steht Ihnen im Navigator ein neuer Eintrag »Programm« zur Verfügung. Hier wird Ihnen eine elektronische Programmzeitschrift angezeigt, über die auch die Aufnahmeplanung erfolgt.

Mit den Tabs im unteren Bereich der Programmzeitschrift wechseln Sie zwischen verschiedenen Tagen hin und her. Die Konfiguration der Programmzeitschrift öffnen Sie über das Schraubenschlüssel-Symbol, das Sie in der unteren linken Ecke der Übersicht finden.

Folgende Einstellungen stehen Ihnen hier zur Verfügung:

  • Registerkarte »Profil«
    Hier definieren Sie, ob und welche Sendungen per automatischer Aufnahmeplanung aufgezeichnet werden sollen.
  • Registerkarte »Kanäle«
    Hier entscheiden Sie, welche Sender in der Programmzeitschrift angezeigt werden sollen, und legen deren Reihenfolge fest (mit den Schaltflächen: »Nach oben« und »Nach unten«).
  • Registerkarte »Erweitert«
    Hier bestimmen Sie die Ansicht der Programmzeitschrift und treffen spezielle Einstellungen zur Aufzeichnung.

Nähere Informationen zu einer Sendung rufen Sie durch einen Klick auf den zugehörigen Eintrag in der Programmübersicht auf. In dem darauf erscheinenden Fenster können Sie über die Schaltfläche »Sendung aufzeichnen« die Aufnahme planen bzw. direkt anstoßen.

Die äußere linke Spalte der Programmzeitschrift zeigt alle geplanten Aufnahmen an. Diese können per Rechtsklick / Kontextmenü auch wieder entfernt werden. Auf Wunsch können Sie bestimmte Sendungen, z.B. Serien, automatisch aufzeichnen lassen. Rufen Sie in der Programmzeitschrift dazu bei der jeweiligen Sendung das Kontextmenü auf und klicken Sie auf »Titel zur Love-List hinzufügen«. Titel, die Sie zur »Hate-List« hinzufügen, werden von der automatischen Aufnahmeplanung ignoriert, so dass generell keine Aufzeichnung dieser Sendungen stattfindet.

Die Priorität der Aufnahmeplanung ist folgendermaßen gestaffelt (angefangen bei höchster bis hin zu niedrigster Priorität):

  1. Einzelne Sendungen, die direkt angewählt und aufgezeichnet werden.
  2. Sendungen, die per »Love-List« aufgezeichnet werden.
  3. Sendungen, die per automatischer Aufnahmeplanung aufgezeichnet werden.

Aufnahmen werden im Ordner »Multimedia > Video > Aufzeichnungen« gespeichert und sind im David Client abrufbar.

Aufzeichnungen geben Sie wieder, indem Sie in der Eintragsliste im David Client den gewünschten Titel markieren und mit einem Doppelklick die Wiedergabe im David Mediaplayer starten. Über die Schaltflächen in der Symbolleiste sowie über die Zeitleiste auf der rechten Seite lässt sich die Wiedergabe steuern.
