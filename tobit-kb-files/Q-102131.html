Q-102.131
Connector for Lotus Notes: Connector Clientseitig installieren


Frage
Problem Wie wird der Connector for Lotus Notes to DvISE Clientseitig installiert, wenn der Dominoserver kein Windows Betriebssystem ist?
Produkt Connector to Lotus Notes for DvISE
Priorität normal
Effekt normal
Datum 15.11.2005

Antwort
Für die Installation des Connector to Lotus Notes for DvISE auf einem Notes-Client müssen die folgenden Voraussetzungen erfüllt werden:

DvISE Tobit Produkte als Backend
FaxWare oder David ab Version 6.5 müssen auf einem Windows NT/2000 oder Novell NetWare Server installiert werden. Abhängig vom Produkt stehen verschiedene Kommunikationsschnittstellen wie Fax, VoiceMail, TMail, SMS, Computer Telefony Integration usw. zur Verfügung.

Rechte des Benutzers
Der Benutzer, der die Installation durchführt, benötigt folgende Zugriffsrechte:

  • »Server« im Programm Lotus Notes bei einer Installation aus einem Notes-Client.
  • »Alle Rechte« auf das Verzeichnis »DAVID« auf dem DvISE-Server.
Lotus Notes Server
Der Connector to Lotus Notes for DvISE kann auf dem Domino-Server (ab Version 4.6.7) oder auf einem Notes-Client ausgeführt werden.
  • Die Names.nsf des Dominoservers sollte ggf. auf den Notes-Client repliziert werden, damit der Connector die Programmdokumente hinterlegen und aktivieren kann.
  • Achten Sie darauf, das sich keine NOTES.INI eines Dominoserver auf dem Notes Client befindet. Dies bedeutet, dass sich der Befehl "Servertasks" nicht in der NOTES.INI befinden darf.
Für die Installation des Connector to Lotus Notes for DvISE starten Sie das Programm DCNSETUP.EXE.

Nach der Grundinstallation wird Lotus Notes gestartet und die Setup-Datenbank DCNsetup.nsf automatisch aufgerufen. In dieser Datenbank wird die gesamte weitere Installation durchgeführt. Unterhalb von Informationen haben Sie die Möglichkeit, die aktuelle Codehistory und alle Setupkomponenten aufzurufen.

Der nächste Schritt in der Setup-Datenbank ist nun die Notesseitige Installation. Um diese durchzuführen klicken Sie auf »Setup« und der Installations-Wizard wird gestartet. Die folgenden Einstellungen müssen Sie in der Setup-Datenbank durchführen, wenn Sie eine clientseitige Installation des Connectors erreichen wollen:

Setup - Zu installierende Komponenten

Wählen Sie zunächst die zu installierenden Komponenten aus. Die folgenden Optionen stehen zur Verfügung:

  • DvISE Connector for Lotus Notes
    Diese Komponente beinhaltet die Administrationsdatenbank und die vom Connector benötigten Bibliotheken. Die Option sollte normalerweise ausgewählt bleiben. Sie können damit eine neue Connector-Datenbank installieren und eine bestehende Datenbank updaten.
  • DvISE Connector for Lotus Notes Task Manager installieren
    Um die Tasks des DvISE Connectors for Lotus Notes zu verwalten, muss der Taskmanager (DCNCtrl und DCNAddin) installiert werden. Bei Auswahl dieses Punktes wird der Taskmanager installiert bzw. upgedated. Auch diese Option sollte ausgewählt bleiben.
Mailschablone erweitern

Die Installation dieser Komponente ist optional und beeinflusst die Funktion des Connectors nicht. Diese Komponente ermöglicht Ihnen Ihre Standardmailschablone um Benutzerunterstützung für die Nutzung des Connectors zu erweitern.Das Setup erweitert die Mailschablone um einige Einträge und ersetzt die Memo-Maske. Wenn Sie bereits Änderungen an der Memo-Maske vorgenommen haben, dürfen Sie diesen Punkt nicht auswählen. Die Anpassung der Mailschablone wird automatisch durchgeführt. Bei der nächsten (meist nächtlichen) Gestaltungsaktualisierung werden die Änderungen für die Benutzer-Maildatenbanken sichtbar.

Setup - Sprachauswahl

Bei den nächsten Schritten müssen Sie die Sprache für Connector und Standard-Mailschablone auswählen.

Setup - Programmpfade

Hier unterscheidet sich zum erstenmal die Installation von einer Connectorinstallation direkt auf dem Dominoserver. Als Lotus Domino-Server Betriebssystem können Sie hier definieren, ob der Connector auf eine NSERVER.EXE zugreifen kann, oder ob die gesamte Konfiguration und Tasksteuerung auf dem Client ausgeführt werden soll. Wenn Sie als Betriebssystem "Anderes" wählen, wird alles, bis auf die DCN.NSF auf den Notes Client installiert.

Setup-DvISE Connector for Lotus Notes

Hier definieren Sie nun den Domino-Server, auf dem die Connector-Datenbank installiert werden soll. Der Servername ist in der Regel vorgeben und kann bei Bedarf abgeändert werden. Dazu müssen Sie den Pfad und den Dateinamen der Connector-Datenbank angeben. Wenn Sie bereits eine frühere Version des Connectors installiert haben, ist auch dieses Feld vorbelegt und Sie sollten das Kontrollkästchen »Bereits Installierte DvISE Connector for Lotus Notes–Version aktualisieren« aktivieren.

Setup - Mailschablone erweitern

Bei sehr grossen Installationen wird ein Dominoserver als Mailserver und einer als Verwalter für die Maildatenbanken verwendet. Unterscheiden sich diese Dominoserver, kann hier der Mailserver explizit hinterlegt werden.

Wenn Sie das Mail-Plugin installieren möchten, gelangen Sie nun in die Eingabe für die abzuändernde Mailschablone. Geben Sie auch hier wieder den Dominoserver ein, auf dem diese Standard-Mailschablone liegt. Ändern Sie wenn nötig den vorgegebenen Dateinamen der Mailschablone entsprechend ab. Wählen Sie die Version dieser Mailschablone aus.

Nun wählen Sie, welche Erweiterungen in der Mailschablone installiert werden sollen. Mit der erweiterten Adressauswahl haben die Anwender die Möglichkeit bei einem neuen Memo direkt auf das Adressbuch zuzugreifen und zum Beispiel die Faxnummer eines Kontaktes auszuwählen und als Empfänger in das Fax einzufügen. Um via Phone Access auf die Notes-Datenbank, ohne auf das Tobit Archiv System, zuzugreifen, muss die Mailschablone um den Remote Access erweitert werden.

Setup – Installation fertig stellen

Durch »Fertigstellen« werden anschließend die gewünschten Komponenten installiert. Nach erfolgreicher Installation der nötigen Daten fahren Sie mit der Einrichtung und Administration des Connectors fort. Die Administrationsdatenbank des Connectors wird nun automatisch geöffnet.
