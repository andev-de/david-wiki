Q-102.213
DvISE Dateien - DAVID\CLIENTS\WINDOWS


Frage
Problem Welche Bedeutung bzw. Aufgabe haben die Daten im Verzeichnis »DAVID\CLIENTS\WINDOWS«?
Produkt David
Priorität keine
Effekt harmlos
Datum 17.09.2002

Antwort
In dem betreffenden Verzeichnis sind die Programmteile des Tobit InfoCenters hinterlegt. Bei Ausführung des Setup-Programms werden diese standardmäßig in die Programmgruppe »Programme -> Tobit InfoCenter« kopiert.
Datei Erläuterung
DVWIN$.* Resource-Datei des Tobit InfoCenters in verschiedenen Sprachen.
DVWIN.HLP Die Hilfe-Datei zum Tobit InfoCenter.
DVWINHLP.* Resource-Dateien zur Windows Online-Hilfe in verschiedenen Sprachen.
DVEDIT.EXE Tobit Image Editor zum Anzeigen von Telefaxen und Voice-Mails.
DVEDIT$.* Resource-Dateien des Tobit Image Editors in verschiedenen Sprachen.
DVREMIND.EXE Programmcode des InfoCenter Notifiers.
DVREM*.WAV Klang-Dateien für den InfoCenter Notifier.
TOBITLIB.DLL »Dynamic Link Library« mit Windows-abhängigen Funktionen des Tobit InfoCenters.
FAXWARE.DRV Druckertreiber für die Erstellung von Faxdateien aus Windows Applikationen.
UNIDRV.DLL »Dynamic Link Library« von Microsoft für den Support von Standard-Druckern.
NWIPXSPX.DLL »Dynamic Link Library« von Novell für den Datentransport über das NetWare Standardprotokoll.
NWCALLS.DLL »Dynamic Link Library« von Novell für das »Application Programming Interface« der NetWare-Shell.
TOBIT!.* Text-Datei mit Einträgen, die im Empfangsbuch des Tobit InfoCenters als Markierungen verwendet werden können.
APPOINT.PRN Vorlage für Ausdruck der Termine aus dem Tobit InfoCenter
DV4TS.EXE Terminalservererweiterung für den FaxWare-Druckertreiber
DVAPI(32).DLL DLLs zur Programmierschnittstelle API
DVATT.EXE Programm zur Anzeige der eMail-Attachments (öffnet die *.$00 in der Archivstruktur)
*.HLP Es handelt sich hierbei um Hilfe-Dateien zu verschiedenen Programmteilen in mehreren Sprachen.
DVCTL32.DLL Diese Datei legt die Verbindungsart (IP-Connection) vom Client zum Server fest.
DVEXTINF.EXE Anwendung zur Anzeige der "Extended Information", die im Tobit InfoCenter eingesehen werden können, wenn sie bei einer markierten Nachricht über die Menüleiste »Aktionen -> Erweiterte Informationen« auswählen. Sie bekommen auf Wunsch u.a. den HTML-Inhalt, die Original-Nachricht oder den SMTP-Header angezeigt.
DVMBSYNC.DLL Diese Datei ist dafür zuständig, dass ein Mobile Client beim Anmelden an den Server mit diesem synchronisiert.
DVNOTES.EXE Anwendung zur Anzeige der Kommentardateien
DVPGP(7).DLL Programmschnittstelle von DvISE zu PGP (Verschlüsselung von eMails)
DVPOP3.EXE Mit diesem Tool kann ein POP3-Account für ein bestimmtes Archiv eingerichtet werden.
DVRemind.EXE Anwendungsdatei für Tobit InfoCenter Notifier
IP.INI Hier wird die IP-Adresse des Servers für die Clients hinterlegt.
OL2TAS.EXE Anwendungsdatei für das Tool zum Datenimport von Outlook ins Tobit InfoCenter
