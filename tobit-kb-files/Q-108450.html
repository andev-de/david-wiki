Q-108.450
Automation - Ab einem bestimmten Empfangswert einmalig Nachrichten versenden


Frage
Problem Sobald vom DvISE Server ein höherer Wert als »20« von einem bestimmten Busteilnehmer (z.B.: vom EIB-Busteilnehmer 1/2/3 einen Temperaturwert) empfangen wird, soll einmalig eine eMail und gleichzeitig eine SMS an eine jeweils fest definierte eMail-Adresse bzw. Mobiltelefonnummer versendet werden.
Produkt Automation
Priorität keine
Effekt harmlos
Datum 9.06.2006

Antwort
Erstellen Sie zunächst eine neue Regel, wie im KnowledgeBase Artikel Q-107.941 beschrieben. Verwenden Sie in dieser Regel ausschließlich die Bedingung »Adresse gleich 1/2/3« und klicken anschließend auf der Registerkarte »Aktion« auf die Schaltfläche »Neues Script«, um den Editor aufzurufen. Vergeben Sie einen beliebigen Betreff und speichern Sie die Änderungen ab. Wechseln Sie nun in das Archive »Scripte« und rufen die soeben erstellte Nachricht auf. Fügen Sie anschließend die nachfolgenden Befehle in das Nachrichtenfeld ein und speichern die Änderungen unter Berücksichtigung der gewünschten Empfängerdaten (eMail-Adresse und Mobiltelefonnummer) erneut ab.

Weiterhin ist es erforderlich, eine Nachricht im Archive »State« zu erstellen, welche als Merker fungiert. Klicken Sie dazu im David InfoCenter auf »Neu«, vergeben einen beliebigen Betreff (hier: "Merker") und speichern diese Nachricht im Archive »State« ab. Auch der vergebene Betreff ist im nachfolgendem Script entsprechend anzupassen.

if (parseInt(m_rxsr.value) >= 20)

Weiterhin werden in die Variable »script« verschiedene DvISE Befehl geschrieben, wie z.B. der DvISE Befehl »@@DATENFORMAT EMAIL@@«. Abschließend ist es erforderlich, den Status des Merkers einmal am Tag auf "0" zurückzusetzen. Dies kann durch die Verwendung eines Timers realisiert werden. Beachten Sie dahingehend die grundlegenden Informationen im KnowledgeBase Artikel Q-107.937. Fügen Sie die nachfolgenden Befehle ein und geben in der Timerserie einen täglichen Zeitpunkt an, an dem der Timer ausgeführt werden soll. Zum Beispiel täglich um 00:00 Uhr. Speichern Sie die Änderungen ab und schließen den Editor.

var sr = get("Merker");
var value = sr.Value+"";

if(value != '0')

{
sr.To = 0;
sr.Value = 0;
sr.save();
}
