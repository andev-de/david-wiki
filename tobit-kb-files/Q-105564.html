Q-105.564
Die Medien-Dienste von David Home


Frage
Problem Was ist unter Medien-Diensten zu verstehen und wie sind diese nutzbar?
Produkt David.InfoCenter PC
Priorität keine
Effekt harmlos
Datum 10.01.2005

Antwort
Durch Nutzung der Medien-Dienste stellt David Home für Sie Ihr individuelles Fernsehprogramm zusammen: Rund um die Uhr und ganz automatisch nimmt David Home für Sie alle Fernsehsendungen auf, die Sie interessieren könnten. Zur Erstellung der Aufnahmeplanung wird ein persönliches Profil verwendet, dass Sie selbst bestimmen können. Die aufgezeichneten Fernsehsendungen werden in speziellen Multimedia Archives innerhalb des Tobit Archive Systems (TAS) abgelegt, von wo aus sie jederzeit direkt abspielbar sind. In diesen Archiveskönnen neben Videos auch beliebige andere Multimedia-Dateien (Bilder, Musiktitel etc.) abgelegt und über das Tobit InfoCenter verwaltet werden.

Beachten Sie, dass die Medien-Dienste nur verfügbar sind, wenn David Home auf einem Computer installiert ist, der unter Windows XP betrieben wird und mit einer geeigneten Hardware für den TV-Empfang über Kabel oder digitale Satelliten-Empfangsanlage ausgestattet ist.

Das aktuelle Fernsehprogramm steht als Online-Programmzeitschrift direkt im Tobit InfoCenter zur Verfügung. Dafür sorgt das Modul »TVIndex«, das als eigenständiger Dienst unter Windows geladen wird. Voraussetzung hierfür ist eine Internet-Anbindung, über die in regelmäßigen Abständen die neuesten Programminformationen eingeholt und direkt in die Online- Programmzeitschrift eingepflegt werden. So ist sichergestellt, das diese Informationen stets dem aktuellen Stand entsprechen und auch kurzfristige Programmänderungen berücksichtigt werden können. Dieser Dienst ist kostenpflichtig und wird über Ihren Account im Club Tobit abgerechnet.

Generell bestehen die TV-Dienste von David Home aus zwei Bausteinen: Zum einen aus der Software selbst und zum anderen aus einer Service-Komponente. Diese Service-Komponente, die ein eigenes Redaktionsteam von Tobit Software bereitstellt, umfasst neben der Online-Programmzeitschrift auch die Lieferung von Zusatzinformationen. Dazu gehört beispielsweise die Gliederung von Spielfilmen in Kapitel, zwischen denen sich hin- und herspringen lässt, oder das Markieren von Werbeblöcken durch so genannte »Stream Tags«, die ein automatisches Überspringen der Werbung beim Abspielen erlauben.

Für die Wiedergabe der aufgezeichneten Sendungen steht Ihnen der InfoCenter Media Player zur Verfügung, der neben Videos auch Musik-Dateien verschiedener Formate abspielen kann. Die Verwaltung Ihrer Medienbibliothek geschieht komplett im Tobit Archive System. Hierzu wird die Archivestruktur unterhalb des Archives mit dem Namen »Medien-Objekte« verwendet. Dort befinden sich Sub Archives für die verschiedenen Medientypen. Natürlich ist diese Struktur bei Bedarf auch erweiterbar.

WDM Capture Port
Um die Medien-Dienste nutzen zu können, muss auf dem David Server ein entsprechender CSS-Port (WDM Capture Port) installiert sein. Wenn zum Zeitpunkt der Installation von David Home bereits eine für David Home zertifizierte TV-Karte (z.B. tvBoard von Tobit) eingebaut und betriebsfähig war, wurde der WDM Capture Port bereits automatisch installiert. Sie finden dann einen entsprechenden Eintrag im DvISE Administrator unterhalb von »Konfiguration -> Ports«. Trifft dies nicht zu, müssen Sie diesen Port manuell hinzufügen.Beachten Sie dabei, dass für den Betrieb des WDM Capture Ports eine freieCSS-Port-Lizenz erforderlich ist.

Der Lieferumfang von David Home beinhaltet zwei CSS-Port-Lizenzen. Es können also entweder zwei Telefon/Fax Ports oder je ein Telefon/Fax Port und einWDMCapture Port gleichzeitig betrieben werden.

Zur Einrichtung des Ports wechseln Sie im DvISE Administrator in den Hauptmenüpunkt »Konfiguration -> Ports« und klicken dann auf das Symbol »Neu«. Daraufhin wird das CSS-Port-Setup gestartet. Klicken Sie hier auf »Hinzufügen«, um einen neuen Port einzurichten oder auf »Überschreiben«, wenn Sie einen bereits bestehenden Port ersetzen wollen.

Wählen Sie in den anschließenden Dialogen die gewünschten Einträge aus der Listeder verfügbaren Hardware sowie aus der Liste der Treiber-Software aus undschließen Sie die Port-Installation ab.

Konfiguration
Um in den Konfigurationsdialog des WDM Capture Ports zu gelangen, öffnen Sie imDvISE Administrator unter dem Menüpunkt »Konfiguration -> Ports« den zugehörigenEintrag, indem Sie darauf doppelklicken.Die Registerkarte »Audio/Video« enthält Einstellmöglichkeiten für die Nutzungder Medien-Dienste. So haben Sie hier Zugriff auf die aktuell genutzte Kanalliste und können diese bei Bedarf bearbeiten.

Sollten beispielsweise bestimmte Kanäle für Sie persönlich generell nicht von Interesse sein, haben Sie hier die Möglichkeit, diese direkt aus der Kanalliste zu entfernen. Über die Schaltfläche »Einstellungen« öffnen Sie ein weiteres Dialogfenster, in dem grundlegende Einstellungen des WDM Capture Ports vorgenommen werden können. Nähere Informationen hierzu finden Sie in der Online Hilfe, die Sie durch einen Druck auf die Taste F1 aufrufen können.

TVIndex
Für die Beschaffung der aktuellen TV-Programminformationen und die Aufbereitung der Online-Programmübersicht, die im Tobit InfoCenter zur Verfügung steht, ist ein eigenständiges Programm-Modul mit dem Namen »TVIndex« zuständig. Wenn Sie bei der Installation von David Home die Nutzung der Medien-Dienste aktiviert haben, ist dieses Modul automatisch als Windows Dienst gestartet. Um den TVIndex manuell zu starten oder zu stoppen, klicken Sie im DvISE Administrator auf die Schaltfläche »DvISE Dienste« und nehmen hier für den Listeneintrag»TVIndex« die gewünschte Einstellung vor.

Die Nutzung des TVIndex erfordert eine Verbindung zum Internet. Um kurze Übertragungszeiten zu gewährleisten, sollte diese Verbindung möglichst schnell sein (z.B. DSL).

Die Informationen zum TV-Programm werden vom Modul TVIndex in regelmäßigen Abständen von einem eigens dafür eingerichteten Server bei Tobit Software abgefragt. Dabei geschieht der Verbindungsaufbau über das Internet. Direkt nach dem ersten Start des Moduls wird diese Verbindung hergestellt und es erfolgt einen einmalige Übertragung sämtlicher verfügbaren Programminformationen. Aufgrund der Datenmenge kann dieser Vorgang einige Zeit in Anspruchnehmen, so dass die TV-Programmübersicht im Tobit InfoCenter nicht sofort zur Verfügung steht. Bei allen folgenden Verbindungen werden dann lediglich Änderungen und Neuerungen abgerufen, was deutlich schneller geschieht.

Dieser Service ist kostenpflichtig und wird über den Club Tobit Software, dem Kundenforum von Tobit Software, abgerechnet. Weitere Informationen zur Abrechnung und Einblick in Ihr Kundenkonto erhalten Sie im Club Tobit Software unter club.tobit.com.

Konfiguration
Die Konfiguration des TVIndex geschieht im DvISE Administrator. Wählen Sie hier im Konfigurationsbaum den Eintrag »TVIndex -> System«. Daraufhin erscheint der Konfigurationsdialog des Moduls.

Hier können Sie unter anderem Einstellungen zu den regelmäßigen Verbindungen mit dem Server bei Tobit Software vornehmen. Bestimmen Sie den Intervall und einen zu berücksichtigenden Zeitrahmen für die regelmäßigen Abfragen. Außerdem können Sie festlegen, dass beim Import der Programminformationen die Übertragung von Bildern nicht stattfinden soll. Auf diese Weise reduziert sich die zu übertragende Datenmenge, allerdings werden dann in der Programmübersicht im Tobit InfoCenter natürlich auch keine Bilder angeboten.
