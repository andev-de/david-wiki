Q-108.972
Module des David Servers


Frage
Problem Welche Module (Dienste) stehen auf dem David Server zur Verfügung und welche Funktionen haben die einzelnen Module?
Produkt David
Priorität normal
Effekt normal
Datum 1.09.2008

Antwort
Der David Server setzt sich aus unterschiedlichen Modulen zusammen, die jeweilsfür bestimmte Aufgabenbereiche zuständig sind. Im einzelnen handelt es sich dabei um folgende Module:

Service Layer
Das Hauptmodul des David Servers. Der David Service Layer koordiniert nahezusämtliche Dienste aller David Module und muss daher stets geladen sein.

PostMan
Das SMTP-Modul von David. Der David PostMan ist ein vollständiger eMailServer für den Austausch von eMails über das SMTP-Protokoll.

Grabbing Server
Der David Grabbing Server ermöglicht den Empfang von eMails über das POP3Protokoll. Dieses Modul dient dazu, externe Postfächer, die sich im Normalfallbei eMail-Providern befinden, abzufragen und die auf diese Weise empfangeneneMails im Tobit Archive System (TAS) auf dem David Server abzulegen

WebBox
Die David WebBox ist ein eigenständiger Web-Server. Sie dient einerseits dazu,Informationen, Nachrichten und Daten, die im Tobit Archive System (TAS) aufdem David Server gespeichert sind, auch per Fernzugriff verfügbar zu machen,andererseits ermöglicht die WebBox außerdem den Aufbau und die Pflege dereigenen Internet-Präsenz.

TVIndex
Das Modul TVIndex sorgt für die Erstellung einer digitalen Fernseh-Programmzeitschrift.Wenn Ihr David Server für den Fernseh-Empfang eingerichtet ist, könnenSie mit Hilfe dieser digitalen Programmzeitschrift eine komfortable automatischeAufnahmeplanung realisieren. Hinweise zum TVIndex entnehmen Sie dem KnowledgeBase Artikel Q-107.986.

ClipInc.
ClipInc. ist ein Modul für das automatische Aufzeichnen von Radioprogrammenund die Bereitstellung so genannter Audio-Clips, die ebenfalls automatischaus dem Radioprogramm extrahiert werden. Informationen zur Nutzung und Konfiguration von ClipInc. im David.Administrator entnehmen Sie dem KnowledgeBase Artikel Q-107.599.

    Hinweis
    Die in David integrierte ClipInc Version wird ab dem Feature Pack 04-Mar-2008 nicht mehr unterstützt. Bei der Installation des Feature Packs 04-Mar-2008 werden evtl. gestartete David ClipInc Dienste auf »Manuell« gestellt und nach dem Update nicht wieder gestartet.

    Zudem steht nach der Installation des FeaturePack der Schalter »Audio Tagging bei StreamTags nutzen« in den Backline Features Konfiguration des David.Administrator nicht mehr zur Verfügung.

PBXpense
PBXpense ermöglicht eine automatische Verarbeitung aller Daten, die von einer Telefonanlage bereitgestellt werden. Die so gewonnenen Informationen können für individuelle Auswertungen genutzt werden. Informationen zur Nutzung und Konfiguration von PBXpense entnehmen Sie dem KnowledgeBase Artikel Q-107.598.

Show Interface Services
Die Show Interface Services stellen die Video-, Audio- und Bilddateien aus den Multimedia Archives von David (Archive/Multimedia) in einem definierbaren Publikationsverzeichnis zur Verfügung. Informationen zur Nutzung und Konfiguration der Show Interface Services entnehmen Sie dem KnowledgeBase Artikel Q-108.462.

Mail Access Server
Das Modul Mail Access Server ermöglicht den Zugriff auf die David Archives mit konventionellen eMail Clients sowohl über das POP3- als auch über das IMAP4-Protokoll. Weitere Informationen zur Konfiguration und Betrieb des Mail Access Server entnehmen Sie dem KnowledgeBase Artikel Q-103.174.

Video Capture
Die Video-Dienste bieten die Möglichkeit, Videoaufzeichnungen über Kameras vorzunehmen und diese im Tobit Archive System (TAS) zu verwalten. Auf diese Weise können z.B. Aufzeichnungen von Sicherheitskameras erstellt und zentral auf dem David Server abgelegt werden. Hinweise zur Voraussetzungen der Video-Dienste entnehmen Sie dem KnowledgeBase Artikel Q-107.129.

Replica
Die Replication Services for David ermöglichen einen Abgleich des Inhalts beliebiger Archives, die sich auf unterschiedlichen David Servern befinden.

Discussion Server
Mit dem Discussion Server stellt David einen kompletten News Server (NNTP Server) zur Verfügung, der es erlaubt, eigene Newsgroups zu führen und zu pflegen und diese per Internet oder Intranet zu publizieren.

Host
David Host dient zum Aufbau eines Mail- und Web-Hosts, der es wiederum ermöglicht, als sogenannter »Internet Hosting Services Provider« (IHS Provider) aufzutreten und Internet-Dienstleistungen für sogenannte IHS Kunden anzubieten, sofern eine Standleitung zum Internet zur Verfügung steht.
