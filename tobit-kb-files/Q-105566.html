Q-105.566
Aufnahme von TV-Sendungen im David InfoCenter


Frage
Problem Wie können TV-Sendungen im David InfoCenter aufgenommen werden?
Produkt David.InfoCenter PC
Priorität keine
Effekt harmlos
Datum 6.02.2006

Antwort
TV-Programmübersicht
Das zentrale Element für die Aufnahmeplanung ist die TV-Programmübersicht. Dabei handelt es sich um eine elektronische Fernsehzeitschrift, die sich in Aussehen und Handhabung an gedruckten Fernsehzeitschriften orientiert. Entscheidend ist, dass Sie diese Zeitschrift nicht jede Woche neu kaufen müssen, sondern dass sie über das Internet aktualisiert wird. So finden auch kurzfristige Programmänderungen den Weg in die aktuelle Programmübersicht.

Sie finden Ihr TV-Programm im Archive-Baum des Tobit InfoCenters unter dem Menüpunkt »Programmübersicht«. Diese Programmübersicht enthält spaltenweise das Programm der ausgewählten Fernsehsender, jeweils in Stundenblöcken abgesetzt und farbig hinterlegt. Beim Öffnen wird die Ansicht zeitlich so eingestellt, dass Sie das gerade laufende Fernsehprogramm im Blick haben. Durch den Schieberegler unterhalb der Programmübersicht können Sie horizontal durch die Sender scrollen.

Die Titelzeile der Programmübersicht enthält neben dem Wochentag und dem zugehörigen Datum je eine Schaltfläche, um zum vorigen bzw. nächsten Wochentag zu wechseln. Über den Menüpunkt »Aufnahmeplanung« finden Sie eine Übersicht der zur Aufnahme vorgesehenen Sendungen, deren Kurzform Sie in der ganz linken Programmspalte sehen. Mit »Einstellungen« können Sie Ihr Benutzerprofil für die TV-Aufzeichnungen und das Aussehen der Programmübersicht beeinflussen. Die beiden letztgenannten Optionen werden in den folgenden Abschnitten behandelt.

Sendungen aufzeichnen
Sie können eine Sendung zum einen vollständig »von Hand« für die Aufnahme programmieren, eine bestimmte Sendung aus der TV-Übersicht für die automatische Aufnahme auswählen oder es einem »intelligenten« Aufnahmeprofil überlassen, die für Sie interessanten Sendungen in Ihrem Auftrag aufzuzeichnen.

Manuelle Aufzeichnung
Bei der manuellen Aufzeichnung stellen Sie sämtliche Aufnahmeparameter von Hand ein. Gehen Sie dazu folgendermaßen vor:

  1. Markieren Sie im Tobit Archive System das Archive »Programmübersicht« und klicken Sie anschließend in der Symbolleiste auf »Neu«. Es öffnet sich das Dialogfenster »Programmaufnahme«.
  2. Vergeben Sie einen Betreff für die Aufnahme oder wählen Sie einen Betreff aus der Liste, indem Sie den kleinen Pfeil neben dem Eingabefeld anklicken.
  3. Tragen Sie bei Bedarf einen Kommentar zur aufzunehmenden Sendung ein.
  4. Geben Sie in das Feld »Kanal« den Namen den Kanals ein, von dem die Sendung aufgezeichnet werden soll. Wählen Sie alternativ einen Kanal aus der Liste.
  5. Setzen Sie den Schalter »Sofort«, wenn unmittelbar mit der Aufnahme begonnen werden soll. Ist dieser Schalter deaktiviert, bestimmen Sie Datum und Uhrzeit für die Aufnahme.
  6. Wählen Sie unter »Dauer« aus der Liste eine Zeitspanne, wie lange das System aufnehmen soll.
  7. Mit einem Mausklick auf »Aufnehmen« wird der Aufnahme-Auftrag erstellt und je nach Einstellung sofort oder zum festgelegten Zeitpunkt gestartet. Sie finden den Auftrag im Versandbuch des Tobit InfoCenters.
Komfort-Aufzeichnung
Die Komfort-Aufzeichnung macht es sehr einfach, aus der aktuellen Programmübersicht eine Sendung für die Aufnahme vorzusehen:
  1. Markieren Sie im Tobit Archive System das Archive »Programmübersicht« und wechseln Sie über die Schaltflächen »Vorheriger Tag« oder »Nächster Tag« in der Titelzeile zum gewünschten Tag, an dem die Sendung ausgestrahlt wird, die Sie aufnehmen möchten.
  2. Klicken Sie mit dem Mauszeiger auf den Namen der Sendung, die Sie aufnehmen möchten. Es öffnet sich ein Dialogfenster mit dem Sendungsnamen in der Titelzeile.
  3. Klicken Sie auf die Schaltfläche »Sendung aufzeichnen«. Daraufhin öffnet sich das gleichnamigeDialogfenster. Hier können Sie weitere Einstellungen für die Aufnahme vornehmen (s. Dialogfenster »Sendung aufzeichnen«). Bestätigen Sie Ihre Eingaben mit »OK«, um die Aufnahmeplanung abzuschließen. Sie finden den Aufzeichnungsauftrag im Versandbuch des Tobit InfoCenters.
Dialogfenster »Sendung aufzeichnen«
In diesem Dialogfenster können Sie einige Optionen für die Aufzeichnung festlegen:
  • Erweiterung des Aufnahmezeitraums
    Wenn die Aufnahme einige Minuten vor dem offiziellen Beginn gestartet werden soll, geben Sie unter »Vorlaufzeit« die gewünschte AnzahlMinuten ein. Ein Standardwert kann in den Grundeinstellungen für TV-Aufzeichnungen eingestellt werden.

    Mit dem Wert »Nachlaufzeit« bestimmen Sie, wie lange eine Sendung nach dem offiziellen Ende noch weiter aufgezeichnet wird. Wählen Sie aus der Liste neben dem Eingabefeld aus, ob die eingetragene Zahl sich auf eine absoluteZeitangabe »in Minuten« bezieht, oder auf einen relativen Wert in »% der Dauer«. Auch hier kann in den Grundeinstellungen für TV-Aufzeichnungen ein Standardwert eingestellt werden.

  • Sendungen mit diesem Titel generell in die automatische Aufnahmeplanung übernehmen
    Setzen Sie diesen Schalter, wenn diese Sendung generell aufgezeichnet werden soll. So verpassen Sie z.B. nie Ihre Lieblingsserie.

  • Titel vor der Übernahme in die Aufgabenplanung editieren
    Wenn Sie den vorigen Schalter gesetzt haben, können Sie mit dieser Option veranlassen, dass im Anschluss ein weiteres Dialogfenster geöffnet wird, in dem zusätzliche Einstellungen vorgenommen werden können.
Aufzeichnungsprofil
Mit dem »Aufzeichnungsprofil« besitzen Sie eine automatische Aufnahmeplanung, die nach Ihren Vorgaben das gesamte Fernsehprogramm aller ausgewählten Sender Tag und Nacht durchsucht und bewertet. In eine Favoriten-Liste können Sie individuelle Schlüsselwörter eintragen, nach denen in den Titeln der TV-Sendungen gesucht wird (z.B. »heute«), wobei auch Teilbegriffe berücksichtigt werden. Die gefundenen Sendungen werden mit hoher Priorität aufgezeichnet. Genauso können Sie in eine Ausschluss-Liste Schlüsselwörter eintragen. Diese Sendungen werden dann auf jeden Fall von der Aufnahme ausgeschlossen.

Grundlage für die Bewertung des Fernsehprogramms sind zwei Kriterien. Zum einen das für jede Sendung vergebene Punktesystem zu den Kategorien »Popularität«, »Spielfilm«, »Humor«, »Information«, »Erotik«, »Action«, »Sport« und »Gefühl«. Hier wird von unserem Redaktionsteam eine Bewertung (eins bis sieben) vergeben, die Sie in den Detail-Informationen jeder Sendung finden. Zum anderen Ihre persönliche Bewertung jeder Kategorie, die Sie in den Systemeinstellungen vornehmen können (s.u.). Über die Verknüpfung dieser zwei Kriterien wird festgelegt, welche Sendung tatsächlich aufgezeichnet wird, wenn zwei gleichzeitig ausgestrahlte Sendungen auf verschiedenen Kanälen für die Aufnahme vorgesehen sind.

So stellt David Home Ihr ganz persönliches Fernsehprogramm zusammen, welches zudem – z.B. für Nachrichtensendungen – immer auf dem neuesten Stand gehalten werden kann.

Um ein individuell abgestimmtes Profil festzulegen, verfahren Sie wie folgt:Markieren Sie im Tobit Archive System das Archive »Programmübersicht«.

  • Wählen Sie in der Titelleiste der Programmübersicht »Einstellungen« und wechseln Sie auf die Registerkarte »Schlüsselwörter«.

  • Klicken Sie neben der Favoriten-Liste auf »Hinzufügen«. Es öffnet sich das Dialogfenster »Textbewertung«.

  • Tragen Sie in das Eingabefeld das gewünschte Schlüsselwort oder einen Teilbegriff ein. Die Optionen in diesem Dialogfenster werden auf Seite 35 im Abschnitt zur Registerkarte »Schlüsselwörter« im Einzelnen beschrieben.

  • Klicken Sie auf »OK«, um Ihre Einstellungen abzuschließen. Das Schlüsselwort wird ans Ende der Liste eingefügt. Die Reihenfolge der Schlüsselwörter bestimmt deren Priorität. Um den Stellenwert für einen Begriff zu erhöhen oder zu verringern, markieren Sie ihn und ordnen ihn mit den Schaltflächen »Nach oben« bzw. »Nach unten« an die gewünschte Position.

  • Klicken Sie neben der "Ausschluß Liste" auf »Hinzufügen«, wenn Sie Sendungen mit bestimmten Schlüsselwörtern oder Teilbegriffen von der Aufzeichnung ausschließen möchten. Sind Schlüsselwörter oder Teilbegriffe sowohl in die "Favoriten Liste" als auch in die "Ausschluß Liste" eingetragen, wird die betreffende Sendung nicht aufgezeichnet.

  • Wechseln Sie anschließend auf die Registerkarte »Profil«. Stellen Sie für jede Kategorie mithilfe der Schieberegler eine Position zwischen »unwichtig« und »wichtig« ein, je nachdem wie hoch Sie die jeweilige Kategorie bewerten. Wählen Sie aus der Liste unter »Automatismus«, wie mit dem Profil verfahren werden soll. Die Auswahl »Diese Einstellungen nicht beachten« ignoriert das Profil vollständig, »Nur besonders hohe Bewertungen beachten« berücksichtigt nur die Einstellungen, bei denen der Schieberegler mindestens auf der vierten Position steht und »Alle Bewertungen beachten« berücksichtigt alle Kategorien so, wie Sie von Ihnen eingestellt wurden. Klicken Sie auf »OK«, um Ihr Aufzeichnungsprofil zu speichern.
Ihre Einstellungen werden sofort wirksam.Die Spalte »Aufnahmeplanung« in der Programmübersichtfüllt oder ändert sich entsprechend den von Ihnen gemachten Bewertungen mit den Sendungen, die aufgezeichnet werden. Zusätzlich zu Titel, Startzeitpunkt und Lauflänge der Sendung weisen Symbole auf die Aufnahmeart hin. Der gelbe Stern besagt, dass die betreffende Sendung aufgrund der von Ihnen eingetragenen Schlüsselwörter aufgenommen wird, der blaue Punkt repräsentiert das Bewertungsprofil und das Personen-Symbol eine von Ihnen manuell eingestellte Aufnahme.

In der Programmspalte der Fernsehkanäle werden die für die Aufnahme vorgesehenen Sendungen mit roter Schrift dargestellt. Mit einem roten Balken hinterlegte Einträge weisen darauf hin, dass die Sendung von Ihnen manuell für die Aufnahme eingestellt wurde.

Schlüsselwörter
Wenn Sie die Sendungstitel nach ganz bestimmten Schlüsselwörtern durchsuchen lassen möchten, um die so gefundenen Sendungen aufzuzeichnen oder von der Aufzeichnung auszuschließen, öffnen Sie in der TV-Programmübersicht aus der Menüzeile die Schaltfläche »Einstellungen« und wechseln zur Registerkarte »Schlüsselwörter«. Klicken Sie anschließend auf »Hinzufügen«.

  • Anzuwendende Zeichenfolge im Titel
    Geben Sie in dieses Eingabefeld das Schlüsselwort oder den Teilbegriff ein, nach dem die Sendungstitel durchsucht werden sollen.

  • Exakte Übereinstimmung der Zeichenfolge
    Wenn Sie diesen Schalter setzen, wird die oben eingetragene Zeichenfolge exakt mit dem Sendungstitel verglichen und nur bei genauer Übereinstimmung als Treffer gekennzeichnet. D.h., wenn Sie z.B. als Zeichenfolge »heute« eintragen, wird eine Sendung nur dann aufgenommen, wenn der Sendungstitel exakt »Heute« heißt. Die Sendung »Heute-Journal« wird demzufolge nicht aufgenommen.

  • Zeitfenster einschränken
    Tragen Sie hier einen Zeitraum ein, in dem nach der oben eingetragenen Zeichenfolge in den Sendungstiteln gesucht werden soll, z.B. 12 bis 18 Uhr.

  • Erweiterung des Aufnahmezeitraums
    Informationen zu dieser Option finden Sie weiter oben in diesem Text.

  • Aufgezeichnete Sendung nach x Tagen löschen
    Geben Sie hier an, nach wie viel Tagen eine aufgezeichnete Sendung wieder gelöscht werden soll. So können Sie zum einen Speicherplatz auf Ihrer Festplatte sparen, zum anderen Ihr aufgezeichnetes TV-Programm aktuell halten, indem z.B. Nachrichtensendungen nach wenigen Tagen wieder gelöscht werden, da sie bereits veraltet sind.
Aufnahmeplanung
Neben der in der TV-Programmübersicht angezeigten Spalte »Aufnahmeplanung«, die alle zur Aufzeichnung vorgesehenen Sendungen in einer Kurzform enthält, finden Sie eine ausführliche Aufbereitung dieser Daten, wenn Sie in der Titelzeile der TV-Programmübersicht die Schaltfläche »Aufnahmeplanung« anklicken.

Das zweigeteilte Fenster zeigt im linken Bereich in chronologischer Reihenfolge die vorgesehenen Sendungen mit Kanal, Start-Uhrzeit, Sendungstitel, Ende-Uhrzeit und Dauer der Aufzeichnung. Wenn Sie eine Sendung markieren, bekommen Sie im rechten Bereich des Fensters – dem Vorschau-Bereich – ausführliche Informationen. Neben den schon bekannten Daten sind das zusätzlich die Bewertung des Redaktionsteams bzgl. Popularität, Humor, usw., und je nach Sendung der Untertitel, die Kategorie, eine Inhaltsangabe, die Liste der Hauptdarsteller, möglicherweise einige Szenenfotos und sonstige Informationen (z.B. Regisseur).
