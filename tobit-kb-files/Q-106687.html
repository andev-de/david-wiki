Q-106.687
Konfiguration der »System Services«


Frage
Problem Was ist bei der Aktivierung bzw. Nutzung der »System Services« im Tobit Software Site Management (Club Tobit Software) zu beachten?
Produkt David
Priorität normal
Effekt normal
Datum 13.12.2005

Antwort
Ihr System erlaubt die Nutzung der »System Services«. Mit einem einzigen Klick im Tobit Software Site Management (Club Tobit Account) können Sie wertvolle Zusatzleistungen aktivieren, die Ihr System sicherer, komfortabler und noch leistungsfähiger machen.

Aufgrund der neuartigen RemoteConfig-Technologie werden auch die Einstellungen an der Konfiguration ihres DvISE Systems vollautomatisch vorgenommen.

Freischalten der System Services im DvISE Service Layer:
Unter Win32 und Linux darf im DvISE Administrator unterhalb von "Konfiguration -> System -> Allgemein" die Option"Online-Konfiguration über Club Tobit Software nicht zulassen" nicht aktiviert sein.

Unter Novell NetWare muss im DvISE Service Layer unterhalb von "Konfiguration -> System -> Optionen" die Option "Subscription-Konfiguration" aktiviert sein.

Funktionsweise:
Sobald Sie eine Zusatzleistung der »System Services« im Club Tobit aktivieren, wird eine Konfigurations-eMail ("DvISE configuration message") vom Club Tobit Server an Ihr DvISE System versendet. Hierzu wird die im DvISE Administrator unterhalb von »Konfiguration -> System« eingetragene »Postmaster eMail Adresse« verwendet. Stellen Sie daher sicher, dass die »Postmaster eMail Adresse« über den DvISE PostMan oder DvISE Grabbing Server erreichbar ist.

Werden zum Beispiel die »Virus Protection Services« aktiviert, dann wird der DvISE Service Layer per HTTP eine Verbindung zum seinem Club Tobit Account aufbauen und hier hier die aktivierten System Services abfragen. Im Club Tobit erscheint neben der Bezeichnung »Virus Protection Services« der Zusatz"wird gestartet". Danach werden im DvISE Administrator die Einstellungen die für diesen Dienst erforderlich sind vorgenommen. In diesem Beispiel wird im DvISE Administrator unterhalb von »Konfiguration -> System -> Virenerkennung -> Signatur Update Intervall (Stunden)« ein Wert eingetragen und der DvISE Grabbing Server gestartet, insofern dieser noch beendet sein sollte. Sobald die Konfiguration erfolgreich vorgenommen wurde, wird ein Rückmeldung des DvISE Service Layer per HTTP zum Club Tobit Server generiert. Die Anzeige "wird gestartet" verschwindet daraufhin wieder. Stellen Sie sicher das der DvISE Server über den Port 80 eine Internetverbindung aufbauen kann. Wenn Sie die Funktion nicht weiter nutzen wollen, entfernen Sie den Haken und es erscheint die Meldung "wird beendet".

Abhängig von der Zusatzfunktion der »System Services« werden folgende Einstellungen in Ihrem DvISE System vorgenommen:

»Virus Protection Services«

  • Unterhalb von »Konfiguration -> System -> System -> Virenerkennung -> Signatur Update Intervall (Stunden)« wird ein Wert von "1" eingetragen
  • Auf dieser Registerkarte wird die Funktion »Dateianhänge überprüfen« aktiviert
  • Der DvISE Grabbing Server wird gestartet, insofern dieser vorher beendet war

»Message Identification Services« (nur unter Win32 und Linux)

  • Die Funktion »Spam-Überprüfung für eingehende eMails aktivieren« unterhalb von »Konfiguration -> System -> Spam-Filter« wird aktiviert

»GSM Message Services«

  • Im DvISE Service Layer wird die Funktion »SMS Provider erzwingen« unterhalb von »Konfiguration -> System -> Optionen« aktiviert
  • Unterhalb von »Konfiguration -> Ports -> Dienste« wird der Dienst »SMS-Rundsendung« aktiviert

»TV Index Services« (nur unter Windows XP)

  • Der Dienst »TVIndex« wird gestartet

»Server Locator Services«

  • Die Funktion »Server Locator Services« wird unterhalb von »PostMan -> System -> Erweitert« aktiviert
  • Die Domänen werden unterhalb von »PostMan -> Datenbanken -> Gültige Domain Namen« eingetragen

»Mail Relay Services«

  • Der Tobit Mail Relay Services wird im DvISE Administrator unterhalb von »PostMan -> System -> Erweitert« aktiviert

»TimeSync. Services«

  • Die Zeitsynchronisation »TimeSync. Services« wird im DvISE Administrator unterhalb von »Konfiguration -> System« bzw. »Konfiguration -> System Services -> Allgemein« aktiviert

Hinweise:

  • Weitere Informationen zu Kosten und Abrechnung der einzelnen Zusatzleistungen entnehmen Sie der »System Services« Webseite im Club Tobit.
  • Die System Services sind bei einer FaxWare Installation nicht im Club Tobit Software aktivierbar.
Die Nutzung der Tobit System Services richtet sich nach dem jeweiligen Server Betriebssystem. Welche System Services bei welchem Betriebssystem genutzt werden können, entnehmen Sie dem KnowledgeBase Artikel Q-107.687.
