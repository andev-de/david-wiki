Q-101.196
Variablen für den DvISE-Server und für das Tobit InfoCenter


Frage
Problem Welche Variablen können für Verteilregeln, Sammellisten oder Weiterleitungen verwendet werden?
Produkt David
Priorität normal
Effekt normal
Datum 1.07.2002

Antwort

Variablen für die Verteilung empfangener Nachrichten

Die folgenden Variablen können Sie zur Erstellung von Verteilregeln und Sammellisten sowie für Weiterleitungen verwenden:

Variable Syntax Wert
SUBJECT %(SUBJECT) Betreff
FROM %(FROM) Absenderadresse bzw. CSID/CLI
TO %(TO) Empfängeradresse bzw. Zielnummer
USER %(USER name) DvISE ID des angegebenen Users
GROUP %(GROUP name) DvISE ID der angegebenen Gruppe
ROOT %(ROOT) Pfad zum DvISE Root Verzeichnis
Archive %(ARCHIVE) Pfad zum Archive Root Verzeichnis
MAILNAME %(MAILNAME) Benutzerkennung der eMail Adresse
DOMAIN %(DOMAIN) Domain Name der eMail Adresse
DOCTYPE %(DOCTYPE) Dokument-Typ als Ziffer
PORT %(PORT) Nummer des DvISE Ports
ROUTING %(ROUTING) Verteilkennung der Nachricht
MAPPING %(MAPPING=string) Setzt die Verteilkennung der Nachricht auf einen beliebigen String

Sie haben auch die möglich, Teilstrings aus den betreffenden Variableninhalten zu extrahieren. Dazu können Sie die Startposition und die Endposition bzw. die Anzahl der auszulesenden Zeichen direkt angeben (Nummerierung der Stellen beginnt bei 0). Dabei ist folgende Syntax zu beachten. Bei diesen Beispielen wird jeweils der Inhalt der Variablen FROM ausgewertet. Statt dessen können Sie natürlich auch andere Variablen einsetzen (z.B. SUBJECT oder TO).

Parameter Rückgabewert
%(FROM,0-5) Alle Zeichen von Position 0 bis Position 5
%(FROM,2- ) Alle Zeichen von Position 2 bis zum Ende
%(FROM,4,2) Genau zwei Zeichen beginnend bei Position 4
%(FROM, ,3) Die letzten 3 Zeichen
%(FROM,1 ) Alle Zeichen von Position 1 bis zum Ende

Fehlt die Angabe der Variable, so wird generell die Verteilkennung (Routing ID) ausgewertet.

Weiterhin haben Sie die Möglichkeit, eine bestimmte Zeichenfolge als Start- oder Endposition für das Auslesen eines Teilstrings aus dem Inhalt einer Variablen festzulegen. Diese Zeichenkette müssen Sie dabei in Anführungszeichen (") angeben. Wird sie als erster Parameter übergeben, markiert sie den Anfang des auszulesenden Teilstrings. Erfolgt diese Angabe als zweiter Parameter, so bestimmt sie dessen Ende. Beispiele:

Parameter Rückgabewert
%(FROM,"@",0,3) Die ersten 3 Zeichen des Domain Namens aus der Absenderadresse einer eMail
%(TO,0,"@") Alle Zeichen der Empfänger-Mailadrese bis zum '@' Zeichen

Ein praktisches Beispiel für eine solche Variable ist eine Weiterleitung im Tobit InfoCenter an einen externen Empfänger. Hier wird der Absender standardmässig nicht ersetzt, daher kann es vorkommen, dass Sie bei der Weiterleitung die Fehlermeldung "ungültiger Absender" erhalten. Tragen Sie einen festen Absender ein, kann der Empfänger der Nachricht nicht mehr feststellen, wer der ursprüngliche Absender war. Hier bietet sich eine Include Datei mit folgender Variable an:

@ @From %(FROM,0,"@")@%(DOMAIN)

In diesem Beispiel wird gleich mit 2 Variablen gearbeitet. Mit dem DvISE-Befehl @ @From wir der Absender ersetzt. Mit der Variable %(FROM,0,"@") werden alle Informationen des Absenders bis zum @ verwendet. Das @ nach der ersten Variable wie in jede Absenderadresse übernommen. Die Variable %(DOMAIN) schriebt hinter dem @ die Domäne des Empfängers und kann auch gegen eine statische Domäne ausgetauscht werden.
