Q-108.727
Konfiguration des David.InfoCenter Notifiers


Frage
Problem Wie kann der InfoCenter Notifier konfiguriert werden bzw. welche Einstellungsmöglichkeiten stehen zur Verfügung?
Produkt David.InfoCenter PC
Priorität normal
Effekt normal
Datum 4.11.2008

Antwort
Der InfoCenter Notifier ist die Erinnerungsfunktion des David.InfoCenters. Mit ihm vergessen Sie keinen Termin und übersehen keine eingegangene Nachricht. Er ist Ihr kleiner Helfer im Hintergrund, der Sie optisch und/oder akustisch an fällige Termine erinnert oder Ihnen eingetroffene Nachrichten meldet.

Standardmäßig wird der InfoCenter Notifier zusammen mit dem David.InfoCenter gestartet. Sie können diese Funktion deaktivieren, indem Sie im Hauptfenster des David.InfoCenter unter »Optionen -> Einstellungen« im Menü »Ansicht -> Zusatzfenster« den Schalter »InfoCenter Notifier starten« deaktivieren.

Öffnen Sie die Notifier Konfiguration, um die gewünschten Einstellungen vorzunehmen. Sie erreichen den Konfigurations-Dialog, indem Sie das Symbol des InfoCenter Notifiers im Windows SysTray mit der rechten Maustaste anklicken und aus dem Menü den Eintrag »Konfiguration...« auswählen.

Sollte der Notifier sich dort nicht befinden, klicken Sie im David.InfoCenter in der Menüleiste auf »Optionen -> Einstellungen« und wechseln zur Registerkarte »Ansicht -> Zusatzfenster«. Aktivieren Sie die Option »InfoCenter Notifier starten« und bestätigen Sie mit »OK«. Beenden Sie anschließend das David.InfoCenter und starten es erneut. Nun befindet sich das Symbol des Notifiers im Windows SysTray.

Auf den Registerkarten »Allgemein« und »Archives« können Sie den InfoCenter Notifier individuell anpassen.

Registerkarte »Allgemein«

  • Ereignisliste
    Ereignisse sind eintreffende Nachrichten, fällige Termine oder Wiedervorlagen. Wenn Sie diesen Schalter aktivieren, öffnet sich bei jedem neuen Ereignis eine Liste, die die Betreffzeilen der noch nicht gelesenen Ereignisse anzeigt. Durch Doppelklick auf die jeweilige Betreffzeile wird der Eintrag im zugehörigen Viewer geöffnet. Sollte das David.InfoCenter zu diesem Zeitpunkt noch nicht geladen sein, so wird es nun gestartet.

    Sofern Sie den Schalter »Immer im Vordergrund« setzen, bleibt die Ereignisliste nach dem Öffnen ständig sichtbar und kann nicht durch andere Fenster überdeckt werden. Dieser Schalter ist allerdings nur aktiv, falls Sie zuvor den Schalter »Ereignisliste« ausgewählt haben.

    Aktivieren Sie die Option »Einblendeffekt nutzen« und die Ereignisliste wird beim Öffnen sanft eingeblendet. Auch diese Option steht nur zur Verfügung, wenn der Schalter »Ereignisliste« zuvor aktiviert wurde.

  • Kurzinfo
    Wenn Sie den Schalter »Kurzinfo« auswählen, wird bei Nachrichteneingang in eins der überwachten Archives ein Hinweisfenster eingeblendet, welches den Namen des betreffenden Archives, die Anzahl der neuen oder ungelesenen Nachrichten und Betreffzeile und Absender der letzten eingetroffenen Nachricht anzeigt.

    Sobald Sie dieses Hinweisfenster anklicken, wird die betreffende Nachricht geöffnet. Auch in diesem Fall wird das David.InfoCenter gestartet,falls noch nicht geschehen. Nach einigen Sekunden schließt sich das Hinweisfenster selbsttätig.

    Wenn Sie den Schalter »Sprechblase anzeigen« aktivieren, wird, nachdem sich das Hinweisfenster wieder geschlossen hat, ausgehend vom Notifier-Symbol im Windows SysTray eine Sprechblase angezeigt. Diese enthält neben der Anzahl der neuen Nachrichten sowohl Betreffzeile als auch Absender der neuesten Nachricht. Der Schalter ist nur verfügbar, wenn Sie zuvor »Kurzinfo« aktiviert haben.

  • Blinken der Tastaturleuchtdioden
    Wenn Sie diesen Schalter setzen, blinken bei einem neuen Ereignis die Leuchtdioden rechts oben auf Ihrer Tastatur in bestimmten Abständen.

  • Systemlautsprecher
    Bei Eingang einer neuen Nachricht wird vom Systemlautsprecher ein Signal gegeben.

  • Sound-Datei
    Bei Empfang einer neuen Nachricht wird eine Sound-Datei abgespielt. Die Sound-Datei kann in der Windows-Systemsteuerung eingestellt werden.

  • Aktualisierung
    Geben Sie hier ein Intervall ein, nach dessen Ablauf die zu überwachenden Archives auf neue Ereignisse überprüft werden.

  • Akustische Benachrichtigung
    Tragen Sie in dieses Feld ein Intervall ein, nach dessen Ablauf Sie immer wieder akustisch an ein neues Ereignis erinnert werden (s. Menüpunkt »Systemlautsprecher« bzw. »Sound-Datei«). Erst wenn Sie die neue Nachricht öffnen, verstummt die akustische Benachrichtigung.

  • Kurzinfo schließen nach
    Stellen Sie hier eine Zeit in Sekunden ein, nach der sich das Kurzinfo-Fenster schließen soll.

Registerkarte »Archives«

Auf dieser Registerkarte können Sie die vom Notifier überwachten Archives festlegen bzw. entfernen. Um zum Beispiel ein Archive nur temporär von der Überwachung auszunehmen, können Sie das Häkchen vor dem Eintrag entfernen. Wenn Sie ein in der Liste aufgeführtes Archive aus der Überwachung durch den InfoCenter Notifier löschen möchten, markieren Sie das gewünschte Archive und klicken Sie auf die Schaltfläche »Löschen«.

  • Ausgangsbuch überprüfen
    Bei Auswahl dieses Schalters wird das Ausgangbuch in die Liste der überwachten Archives übernommen. Dadurch werden Nachrichten, die aus irgendeinem Grund nicht versendet werden konnten und im Status-Feld eine Fehlermeldung enthalten, über den Notifier gemeldet.
