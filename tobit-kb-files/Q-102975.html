Q-102.975
Lizensierung und UpDate auf David XL


Frage
Problem Was ist beim Kauf von David XL bzw. beim UpDate zu beachten?
Produkt David
Priorität keine
Effekt normal
Datum 9.12.2002

Antwort
Mit der neuen David-Version von Tobit Software, "David XL", wurde auch ein neues Lizensierungs-System vorgestellt.
Kern des Produktes ist die Tobit Start License, welche die notwendigen allgemeinen Dienste bereitstellt. Diese beinhaltet außerdem eine Full-Port-Lizenz. Diese Startlizenz können Sie durch Tobit Scale Licenses um die gewünschte Anzahl an Benutzern sowie zusätzliche Ports erweitern. Benutzerlizenzen sind in den Größenordnungen 5, 25 und 50 User erhältlich. Jeder zusätzliche Port erfordert eine Portlizenz, damit ein weiterer B-Kanal Ihrer ISDN-Karte angesprochen werden kann. Die Anbindung an EIB (Gebäudeautomation) erfolgt ebenfalls über eine zusätzliche Portlizenz bzw. kann direkt über den vorhandenen Port realisiert werden.
David XL ist betriebsbereit, wenn mindestens die Tobit Start License sowie eine Tobit Scale License mit weiteren Benutzern eingetragen wurde.

Diese Grundstruktur kann jedoch nicht nur um zusätzliche Benutzer und Ports erweitert werden, sondern Tobit Software bietet auch noch Tobit Improvement Licenses sowie Tobit Subscription Licenses.

Tobit Improvement Licenses bieten Ihnen zusätzliche Features, die unbegrenzt nutzbar sind, sobald diese Lizenz eingetragen wurde. Folgende Features können über jeweils eine dieser Lizenzen hinzugefügt werden:

  • Internet Access Module for DvISE
    Dieses Modul ermöglicht den Zugriff auf die persönlichen Daten per Web und WAP.
  • Extended Telephony Module for DvISE
    Erweitert David XL um ein grafisches Programmiertool zur Steuerung von Kommunikationsabläufen, synthetischer Sprachausgabe und Spracherkennung.
  • Optical Character Recognition for DvISE
    Ermöglicht die automatische Texterkennung für das Auslesen von Textinformationen aus Telefaxen.
  • Tobit InfoCenter PDA Synchronisation
    Mit diesem Modul kann der Datenaustausch zwischen Palm-PDA's und dem Tobit InfoCenter realisiert werden.
  • Home Automation Module for DvISE
    Erlaubt die Haussteuerung und -automation mittels des DvISE-Servers.
  • Connector to Microsoft Exchange for DvISE
  • Connector to Lotus Notes for DvISE
  • Connector to Novell GroupWise for DvISE

Die Tobit Subscription License stellt ebenfalls zusätzliche Features bereit. Diese sind jedoch mengen- oder zeitbegrenzt. Als Tobit Subscription License erhalten Sie folgende Module:

  • Virus Protection for DvISE
    Dieses Modul sorgt durch das automatische UpDate von Virensignaturen für aktuellen Virenschutz. Die Laufzeit beträgt 12 Monate.
  • PrePaid SMS Package for DvISE
    SMS Paket mit einem vorbezahlten Kontingent an SMS Nachrichten. Die Pakete sind in den Größenordnungen 500, 1000 und 2000 SMS erhältlich.

Natürlich besteht auch die Möglichkeit für eine vorhandene Installation ein UpDate zu erwerben. Dabei sind einige Dinge zu beachten.

  • Beim UpDate bleiben Ihnen zusätzlich eingetragene Ports erhalten. Hierfür muss nicht extra eine neue Tobit Scale License erworben werden.
  • Anders jedoch bei den Benutzern: Da bei der Tobit Start License keine Benutzer inklusive sind, müssen die vorhandenen Benutzer mittels einer Tobit Scale License eingetragen werden. Bei einem UpDate können diese Scale Licenses jedoch für die Anzahl an bereits vorhandenen Benutzern zu einem günstigen Preis bezogen werden. Bitte beachten Sie, dass Sie mit diesen speziellen Scale Licenses die Anzahl der Benutzer nicht erhöhen können! Um neue Benutzer hinzuzufügen benötigen Sie die Scale Licenses zum normalen Preis.
  • Bei einem UpDate von David Professional 6.6 auf David XL sind die Telephony- und Internet-Funktionen nicht mehr enthalten, können jedoch mit Zusatzlizenzen bei Bedarf hinzugefügt werden.
  • Die synthetische Sprachausgabe sowie die Spracherkennung gehören bei David XL nicht mehr serienmäßig zum Leistungsumfang. Wer den Phone Access nutzen oder TMails versenden möchte, kann diese Funktionen mit dem Erwerb einer Lizenz für das Extended Telephony Module freischalten. Das Extended Telephony Module for DvISE enthält zusätzlich mit dem DvCC eine grafische Programmieroberfläche zur Ablaufsteuerung. Auch der IHS-Host wird in dem Module enthalten sein.
Mit dem neuen Lizensierungssystem kann Ihr DvISE-System somit genau an Ihre Ansprüche und Bedürfnisse angepasst werden.
