Q-100.182
Parameter der Datei »POSTMAN.INI«


Frage
Produkt David PostMan
Priorität keine
Effekt harmlos
Datum 6.10.2005

Antwort
Die folgenden Parameter können Sie in der Datei »POSTMAN.INI«, welche sich im DvISE Server Verzeichnis »\DAVID\APPS\POSTMAN\CODE« befindet, bearbeiten:

SendThreadCount = 5
Default ist 5
Mit der entsprechenden Eingabe einer Anzahl wird nun nach einem Neustart des DvISE Postman bzw. des DvISE Service Layers die Anzahl der SMTP-Ports erhöht.

WatchdogDelay = 55
Default ist 55
Dieser Parameter bestimmt die maximale Wartezeit für ankommende Datenblöcke.

MinDiskSpace = 10
Default ist 10 MB
Dieser Parameter bestimmt den freien Speicherplatz, unter dem die Module dann keine Mails mehr empfangen bis wieder mehr Speicherplatz vorhanden ist.

NoRecipientList = FALSE
Default = FALSE
Bei der Einstellung TRUE werden im SMTP Header keine To und Cc Informationen übertragen

EHLOWITHDOMAIN = FALSE
Default is FALSE
Legt fest, ob DvISE PostMan beim Versand über einen Provider beim SMTP EHLO/HELO Kommando die lokale IP Adresse (FALSE)oder die in der DvISE PostMan Systemkonfiguration ("SMTP Host Name") eingetragene Domain angibt (TRUE).

Hinweis:
Bei Änderungen in den INI-Dateien von David sollte Sie das entsprechende Modul sowie den DvISE Service Layer neu starten. Das vorgestellte Semikolon (;) müssen Sie dazu entfernen, da dieses die entsprechenden Parameter deaktiviert.
