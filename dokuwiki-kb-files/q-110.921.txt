====== Q-110.921 - Wiedervorgelegte Nachricht beantworten ======
~~NOCACHE~~
!!!
==== Problem ====

Wird eine interne Nachricht auf Termin versendet und anschließend auf die wiedervorlegte Nachricht geantwortet, so erhielt die Nachricht den Status "Wiedervorlage" mit dem Zeitpunkt des ursprünglichen Termins. Jetzt wird die aktuelle Zeit verwendet.
