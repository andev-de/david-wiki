====== Q-110.024 - Dateien für Mobile Client ======
==== Problem ====
Beim Update über das Installation Package wurde u.U. die tobit32.dll für die Mobile Installation nicht aktualisiert. Das wird nun beim neuen Installation Package berücksichtigt.
==== Produkt ====
David
==== Datum ====
17.11.2009
==== Antwort ====
Das oben beschriebene Verhalten ist mit dem David.fx Service Pack 17-Nov-2009 behoben. Um eine bestehende Installation auf den aktuellen Entwicklungsstand zu bringen, laden Sie sich das entsprechende Download für Ihre David Version herunter, führen die Download-Datei aus und starten die Installation.

Halten Sie Ihre David-Installation stets auf dem neuesten Stand. Durch regelmäßiges Einspielen der aktuellen Programmversionen und Service Packs stellen Sie sicher, dass Sie Ihre Software stets mit einem Höchstmaß an Sicherheit und Komfort nutzen können. 

Allgemeine Hinweise, Informationen zur Installation sowie die zugehörige Code History finden Sie in der jeweiligen Download-Sektion.
