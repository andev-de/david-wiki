====== Q-110.132 - Signierte eMail nach Serverwechsel versenden ======
==== Problem ====
Wird eine neue signierte eMail erstellt und dann auf einem David Server ohne installiertes Zertifikat gewechselt, so gab es beim anschließenden Versand eine Fehlermeldung. Nun erscheint in dem Fall die Abfrage, ob die Nachricht ohne Zertifikat versendet werden soll.
==== Produkt ====
David Client
==== Datum ====
11.03.2010
==== Antwort ====
Das oben beschriebene Verhalten ist mit dem David.fx Service Pack 11-Mar-2010 behoben. Um eine bestehende Installation auf den aktuellen Entwicklungsstand zu bringen, laden Sie sich das entsprechende Download für Ihre David Version herunter, führen die Download-Datei aus und starten die Installation.

Halten Sie Ihre David-Installation stets auf dem neuesten Stand. Durch regelmäßiges Einspielen der aktuellen Programmversionen und Service Packs stellen Sie sicher, dass Sie Ihre Software stets mit einem Höchstmaß an Sicherheit und Komfort nutzen können. 

Allgemeine Hinweise, Informationen zur Installation sowie die zugehörige Code History finden Sie in der jeweiligen Download-Sektion.
