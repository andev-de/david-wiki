====== Q-110.878 - Schutzverletzung beim Zugriff per david Client Web ======
~~NOCACHE~~
!!!
==== Problem ====

Sind für einen Benutzer keine Zugangsdaten eingetragen und der Benutzer greift per david Client Web auf seine Ordner zu, konnte es zum Absturz der WebBox kommen. Voraussetzung war, dass in der webbox.ini der Parameter "ForceWebAccess=FALSE" gesetzt ist.
