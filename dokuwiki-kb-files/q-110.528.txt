====== Q-110.528 - Termine kopieren (CalDAV) ======
~~NOCACHE~~
!!!
==== Problem ====

Werden Termine per CalDAV synchronisiert, wurde das Kopieren von Terminen per David Client bisher nicht korrekt berücksichtigt. Das ist mit dem neuen David Client nun behoben.

==== Produkt ====

David Client

==== Datum ====

3.05.2011

==== Antwort ====

Die oben beschriebene Aktualisierung bzw. die Korrektur des oben beschriebenen Verhaltens ist im David.fx Edition 2011 Service Pack 03-Mai-2011 enthalten. Um eine bestehende Installation auf den aktuellen Entwicklungsstand zu bringen, laden Sie sich das entsprechende Download für Ihre David Version herunter, führen die Download-Datei aus und starten die Installation.

Halten Sie Ihre David-Installation stets auf dem neuesten Stand. Durch regelmäßiges Einspielen der aktuellen Programmversionen und Service Packs stellen Sie sicher, dass Sie Ihre Software stets mit einem Höchstmaß an Sicherheit und Komfort nutzen können. 

Allgemeine Hinweise, Informationen zur Installation sowie die zugehörige Code History finden Sie in der jeweiligen Download-Sektion.
