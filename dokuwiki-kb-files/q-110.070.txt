====== Q-110.070 - Ausnahmen bei Terminserien ======
==== Problem ====
Beim Öffnen eines Elements einer Terminserie erscheint nun ein Abfrage, ob das Element oder die Serie geöffnet werden soll. Wird das Element geöffnet, kann es unabhängig von der Serie geändert werden (z.B. andere Tageszeit), ist aber weiterhin mit der Serie verknüpft. Auch beim Verschieben eines Elements einer Serie in der Tages- oder Arbeitswochenansicht kommt in dem Fall nun eine Abfrage.
==== Produkt ====
David Client
==== Datum ====
7.01.2010
