====== Q-110.059 - Aktualisierung ViProtect ======
==== Problem ====
ViProtect wurde aktualisiert und enthält nun das Patch 2 von "McAfee VirusScan Enterprise 8.7i". U.A. entfällt damit die Meldung "Der Status wird in einem nicht mehr unterstützen Format an das Windows Sicherheitscenter gemeldet" unter Windows 7.Wurde ViProtect in einer Gruppenrichtlinie als MSI Paket vom Administrator verteilt, so ist es notwendig das vorhandene VSI870.MSI Paket zu entfernen und in diesem Zuge einen Eintrag für das neue Paket VSI871.MSI hinzuzufügen.
==== Produkt ====
Tobit.ViProtect
==== Datum ====
7.01.2010
==== Antwort ====
Das oben beschriebene Verhalten ist mit dem David.fx Service Pack 07-Jan-2010 behoben. Um eine bestehende Installation auf den aktuellen Entwicklungsstand zu bringen, laden Sie sich das entsprechende Download für Ihre David Version herunter, führen die Download-Datei aus und starten die Installation.

Halten Sie Ihre David-Installation stets auf dem neuesten Stand. Durch regelmäßiges Einspielen der aktuellen Programmversionen und Service Packs stellen Sie sicher, dass Sie Ihre Software stets mit einem Höchstmaß an Sicherheit und Komfort nutzen können. 

Allgemeine Hinweise, Informationen zur Installation sowie die zugehörige Code History finden Sie in der jeweiligen Download-Sektion.
