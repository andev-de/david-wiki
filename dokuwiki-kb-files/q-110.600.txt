====== Q-110.600 - Eigene Feeds zur Auswahl erstellen ======
~~NOCACHE~~
!!!
==== Problem ====

Unter \david\code\common\rss-ico kann nun eine Datei feeds.xml erstellt werden. Die enthaltenen Einträge werden bei den David Clients zur Auswahl gestellt (im Dialog "Feed hinzufügen" beim neuen Reiter "Intern"). Eine Beispiel Datei wird mitgeliefert.

Die Parameter sind:name - Name des Feedsurl - Url des Feedsquickinfo - Beschreibender Text zur Urltypeid - Feed Typ 0 = Text, 1 = Video, 2 = Soundimageurl - Bild für die Vorschau (falls gewünscht)lang - Sprache des Feeds (GER/ENG/FRA etc.)herausgeber - Provider des Feeds.

==== Produkt ====

David Client

19.08.2011
