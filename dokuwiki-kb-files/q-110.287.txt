====== Q-110.287 - Volltextsuche in Kommentar ======
==== Problem ====
Die Volltextsuche kann nun auch im "Kommentar" verwendet werden.
==== Produkt ====
David Service Layer
==== Datum ====
14.10.2010
==== Antwort ====
Das oben beschriebene Verhalten ist mit dem David.fx Edition 2011 Service Pack 13-Oct-2010 behoben. Um eine bestehende Installation auf den aktuellen Entwicklungsstand zu bringen, laden Sie sich das entsprechende Download für Ihre David Version herunter, führen die Download-Datei aus und starten die Installation.

Halten Sie Ihre David-Installation stets auf dem neuesten Stand. Durch regelmäßiges Einspielen der aktuellen Programmversionen und Service Packs stellen Sie sicher, dass Sie Ihre Software stets mit einem Höchstmaß an Sicherheit und Komfort nutzen können. 

Allgemeine Hinweise, Informationen zur Installation sowie die zugehörige Code History finden Sie in der jeweiligen Download-Sektion.
