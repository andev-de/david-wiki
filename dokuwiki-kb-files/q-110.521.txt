====== Q-110.521 - Empfangen von Besprechungsanfragen ======
~~NOCACHE~~
!!!
==== Problem ====

Externe empfangene Besprechungsanfragen (in diesem Fall von Microsoft Office Outlook 12.0 erzeugt) wurden u.U. nicht als solche erkannt und somit standen im David Client nicht die Optionen zum Annehmen/Ablehnen zur Auswahl.

==== Produkt ====

David PostMan

==== Datum ====

3.05.2011

==== Antwort ====

Die oben beschriebene Aktualisierung bzw. die Korrektur des oben beschriebenen Verhaltens ist im David.fx Edition 2011 Service Pack 25-Apr-2011 enthalten. Um eine bestehende Installation auf den aktuellen Entwicklungsstand zu bringen, laden Sie sich das entsprechende Download für Ihre David Version herunter, führen die Download-Datei aus und starten die Installation.

Halten Sie Ihre David-Installation stets auf dem neuesten Stand. Durch regelmäßiges Einspielen der aktuellen Programmversionen und Service Packs stellen Sie sicher, dass Sie Ihre Software stets mit einem Höchstmaß an Sicherheit und Komfort nutzen können. 

Allgemeine Hinweise, Informationen zur Installation sowie die zugehörige Code History finden Sie in der jeweiligen Download-Sektion.
