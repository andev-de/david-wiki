====== Q-110.504 - "gelesene Nachrichten" in InfoCenter Ansicht ======
~~NOCACHE~~
!!!
==== Problem ====

Wird die Anzahl der angezeigten Nachrichten unter "Ansicht/Filter" beschränkt, so wurden in der InfoCenter Ansicht neben ungelesenen Nachrichten u.U. auch gelesene Nachrichten angezeigt.

==== Produkt ====

David Client

==== Datum ====

11.04.2011

==== Antwort ====

Die oben beschriebene Aktualisierung bzw. die Korrektur des oben beschriebenen Verhaltens ist im David.fx Edition 2011 Service Pack 11-Apr-2011 enthalten. Um eine bestehende Installation auf den aktuellen Entwicklungsstand zu bringen, laden Sie sich das entsprechende Download für Ihre David Version herunter, führen die Download-Datei aus und starten die Installation.

Halten Sie Ihre David-Installation stets auf dem neuesten Stand. Durch regelmäßiges Einspielen der aktuellen Programmversionen und Service Packs stellen Sie sicher, dass Sie Ihre Software stets mit einem Höchstmaß an Sicherheit und Komfort nutzen können. 

Allgemeine Hinweise, Informationen zur Installation sowie die zugehörige Code History finden Sie in der jeweiligen Download-Sektion.
