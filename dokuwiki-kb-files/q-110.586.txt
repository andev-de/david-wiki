====== Q-110.586 - Inline Grafiken per Data Protokoll ======
~~NOCACHE~~
!!!
==== Problem ====

Wird in einer HTML eMail für Inline Grafiken das Data Protokoll verwendet (z.B. bei von Thunderbird erzeugten eMails), so stürzte der Client beim Weiterleiten/Beantworten einer solchen eMail u.U. ab. Außerdem bleiben nun beim Speichern von eMails mit Inline Grafiken als HTML Datei diese Grafiken erhalten. Zuvor wurde dort beim Öffnen der HTML Datei nur das rote Kreuz für eine fehlende Grafik angezeigt.

==== Produkt ====

David Client

==== Datum ====

19.08.2011
