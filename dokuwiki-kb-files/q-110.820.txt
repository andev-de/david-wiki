====== Q-110.820 - TeamChat ======
~~NOCACHE~~
!!!
==== Problem ====

Im David Client Web kann nun auch der TeamChat genutzt werden (ab Flash Player Version 10). Hierzu wird zusätzlich der TCP Port 843 verwendet. Sollte dieser Port bereits für andere Applikationen vorgesehen sein, kann die Verwendung über den Parameter ProvideFlashSocket = FALSE in der WebBox.ini deaktiviert werden.

David Client Web
