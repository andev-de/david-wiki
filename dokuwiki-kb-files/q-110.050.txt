====== Q-110.050 - Fax speichern mit dem Image Editor ======
==== Problem ====
Beim Speichern eines Fax mit dem Image Editor wurde u.U. unter Windows 7 bzw. Server 2008 ein angegebener Dateiname nicht übernommen und die Datei somit nur als *.tif gespeichert.
==== Produkt ====
David Client
==== Datum ====
18.12.2009

Das oben beschriebene Verhalten ist mit dem David.fx Service Pack 18-Dec-2009 behoben.
