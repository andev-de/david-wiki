====== Q-110.677 - Zugriff auf Ordner mit SQL Adressen ======
~~NOCACHE~~
!!!
==== Problem ====

Beim Erstellen einer neuen Adresse in einem SQL Ordner oder dem Ändern vorhandener Adressen konnte es zu einer hohen Auslastung durch den Client und den Service Layer kommen. Außerdem wurde bei jedem Zugriff auf einen Ordner mit SQL Adressen eine neue Verbindung zu dem SQL Server erstellt.
