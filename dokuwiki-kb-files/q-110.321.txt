====== Q-110.321 - eMail Weiterleitung (Status "Temporär beschäftigt") ======
==== Problem ====
Ist PostMan für das Weiterleiten eingerichtet und die Gegenstelle liefert den Status "Temporär beschäftigt" zurück, so wurde der Auftrag u.U. mehrmals im Versand eingetragen. Der Auftrag wird nun nur noch einmal eingetragen und die jeweiligen Sendeversuche erhalten den Status "belegt".
==== Produkt ====
David PostMan
==== Datum ====
15.11.2010
==== Antwort ====
Das oben beschriebene Verhalten ist mit dem David.fx Edition 2011 Service Pack 12-Nov-2010 behoben. Um eine bestehende Installation auf den aktuellen Entwicklungsstand zu bringen, laden Sie sich das entsprechende Download für Ihre David Version herunter, führen die Download-Datei aus und starten die Installation.

Halten Sie Ihre David-Installation stets auf dem neuesten Stand. Durch regelmäßiges Einspielen der aktuellen Programmversionen und Service Packs stellen Sie sicher, dass Sie Ihre Software stets mit einem Höchstmaß an Sicherheit und Komfort nutzen können. 

Allgemeine Hinweise, Informationen zur Installation sowie die zugehörige Code History finden Sie in der jeweiligen Download-Sektion.
