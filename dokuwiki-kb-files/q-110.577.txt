====== Q-110.577 - Anzeige neuer Nachrichten ======
~~NOCACHE~~
!!!
==== Problem ====

Ist der Client gestartet und der aktuelle David Server wird über den Menüpunkt "Netzwerk" erneut ausgewählt, so wurde u.U. anschließend die Anzahl neuer Nachrichten nicht mehr aktualisiert.

==== Produkt ====

David Client

==== Datum ====

19.08.2011

==== Antwort ====

Die oben beschriebene Aktualisierung bzw. die Korrektur des oben beschriebenen Verhaltens ist im David.fx Edition 2011 Service Pack 12-AUG-2011 enthalten. Um eine bestehende Installation auf den aktuellen Entwicklungsstand zu bringen, laden Sie sich das entsprechende Download für Ihre David Version herunter, führen die Download-Datei aus und starten die Installation.

Halten Sie Ihre David-Installation stets auf dem neuesten Stand. Durch regelmäßiges Einspielen der aktuellen Programmversionen und Service Packs stellen Sie sicher, dass Sie Ihre Software stets mit einem Höchstmaß an Sicherheit und Komfort nutzen können. 

Allgemeine Hinweise, Informationen zur Installation sowie die zugehörige Code History finden Sie in der jeweiligen Download-Sektion.
