====== Q-110.324 - "Senden und Empfangen" mit Outlook (IMAP) ======
==== Problem ====
Hat man Outlook per IMAP mit dem Mail Access Server eingerichtet, so wurden beim Klick auf "Senden und Empfangen" in Outlook keine neuen Nachrichten geholt, wenn man im Posteingang steht. Man musste erst den Ordner wechseln. Jetzt funktioniert das Abholen neuer Nachrichten auch in diesem Fall.
==== Produkt ====
David Mail Access Server
==== Datum ====
15.11.2010

Das oben beschriebene Verhalten ist mit dem David.fx Edition 2011 Service Pack 12-Nov-2010 behoben.
